瀑布流

@Masonry
概述 Overview
.本意是交错排列的砖石结构，强大的前端开发者形象的称之为瀑布流
.特点：宽度相同，高度不同
.早期多使用 js + float 实现；现在可以使用 flex、column 快速实现
.为调试方便，可以先使用占位图片 Lorem Picsum 开发
<img src="https://picsum.photos/200/300?random=1">
弹性盒子 flex
实现：列方向、换行、固定列宽
优点：方便采用结构伪类编号
不足：高度做不到任性，需调试找到最佳高度 - 刚好放满指定的列，且不能出现水平滚动条；如果使用 gap 还需要精确计算没列宽度
.masonry-flex {
  display: flex;
  flex-direction: column;
  flex-wrap: wrap;
  gap: 10px;
  height: 2400px;
}
.masonry-flex .item {
  width: calc((100% - 30px)/4);
}

@media screen and (max-width:768px) {
  .masonry-flex {
    height: 600px;
  }
}
分栏 Columns
实现：无需指定高度，只需指定列数即可；指定 gap 为0可以紧凑排列
不足：不太方便编号
.masonry-columns {
  column-count: 4;
  gap: 10px;
}

.masonry-columns .item {
  margin-bottom: 10px;
}
