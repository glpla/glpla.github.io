锚点跳转*
说明
. 跳转到当前页面内部或其它页面内部某个元素位置
. 定义：在目标元素中，使用 id 属性定义一个锚点
. 使用：在 <a> 的 href 属性中，使用 # 识别锚点
. 页面返回时不能直接退出当前页面，需要逐级返回；后期多使用 js 实现跳转
1. 定义锚点：给目标元素指定 id 属性充当锚点；W3C规范仅仅允许使用 id 标记锚点，不能使用类.
<div id='cont'>我是目标元素</div>
2. 为<a>的 href 属性指定目标为元素的id名字，配合#，表示是id属性
<a href="#cont"> 去目标元素 </a>

页面跳转*
说明
. 页面之间的跳转
. 根据页面是否属于同一个网站，又可以分：站内跳转和站外跳转
站内跳转
同一个网站内部页面之间的跳转，多使用相对路径
<a href="./greeting.html"> 目标页面 </a>
也可以转跳转到站内其它页面的某个锚点
<a href="./blog.html#cont"> 目标页面某个元素 </a>
站外跳转
跳转到其它网站页面或页面内某个锚点，多使用绝对路径或完整路径
建议使用完整网络协议。否则测试会告警
<a href="https://www.baidu.com"> 百度 </a>
<a href="https://www.baidu.com#baike"> 百度百科 </a>
也可以不指定网络协议，由系统自行判断；不建议
<a href="//www.baidu.com"> www.baidu.com </a>

改善用户体验
1. 添加title属性，完善用户体验
<a href="" title="" target="_blank"></a>
2. 使用CSS目标选择器高亮当前跳转到的锚点|目标
el:target {
  color: #f40;
}
3. 使用CSS为元素的滚动容器指定 滚动行为 - scroll behavior 为丝滑滚动smooth
html {
  scroll-behavior: smooth;
}
4. 使用CSS为元素的滚动容器指定滚动填充scroll-padding-top为导航区的高度
html {
  scroll-padding-top: 100px;
}
5. 滚动容器通常为html或者是某个具体的容器标签元素，该容器必须显式的指定高度且具备溢出隐藏的属性
.wrap {
  height: 100vh;
  scroll-behavior: smooth;
  overflow-y: auto;
}
样式
样式初始化
. 为保证风格统一，通常覆盖<a>的默认样式，重新设计
. normalize.css
. reset.css
以下各例均使用初始化样式
a {
    text-decoration: none;
    color: inherit;
    font: inherit;
}
样式伪类
系统提供了4个默认的 伪类；如果定义样式，需要按照一定的顺序声明
a → a:link → a:visted → a:hover → a:active
a {
  font: inherit;
  color: inherit;
  text-decoration: none;
}

a:link,
a:visited {
  border-bottom: 1px solid #ccc;
}


a:hover,
a:focus {
  border-color: #f40;
  color: #f40;
}

a:active {
  background-color: #f40;
  color: #fff;
}
