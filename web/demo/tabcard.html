标签页

@TabCard
单击标签，显示对应的内容
可以使用纯 CSS 实现，也可以使用 JavaScript 实现
适合少量数据的展示
:target
配合 <a> 使用
初态，各锚点隐藏
锚点成为目标时，显示
加入更多结构和样式的设计
<nav>
  <a href="#home">Home</a>
  <a href="#info">Info</a>
  <a href="#work">Work</a>
  <a href="#team">Team</a>
  <a href="#about">About!</a>
</nav>
<div>
  <div class="nav-target" id="home">Lorem ipsum dolor sit amet.</div>
  <div class="nav-target" id="info">Minima maxime exercitationem cumque voluptates.</div>
  <div class="nav-target" id="work">Facilis, voluptatem? Et, mollitia ratione.</div>
  <div class="nav-target" id="team">Laborum fugiat doloribus tempora consectetur?</div>
  <div class="nav-target" id="about">Sunt, earum? Eius, vitae corrupti!</div>
</div>
.nav-target {
  display: none;
}

.nav-target:target {
  display: block;
}
:checked
配合表单的单选按钮和 <label> 实现
每个 <input> 和 <label> 关联
初态：各 <input> 隐藏，其中一个默认选中
单击某个单选按钮，使用特定的选择器显示对应的内容
注意结构的设计
需要更多样式加持，但是使用比较灵活；可以使用普通选择器、属性选择器或结构伪类选择器
<div>
  <input type="radio" name="tabard" hidden id="home" checked>
  <label for="home">Home</label>
  <input type="radio" name="tabard" hidden id="info">
  <label for="info">info</label>
  <input type="radio" name="tabard" hidden id="work">
  <label for="work">work</label>
  <input type="radio" name="tabard" hidden id="team">
  <label for="team">team</label>
  <input type="radio" name="tabard" hidden id="about">
  <label for="about">about</label>
  <div class="cont">
    <div id="home">Lorem ipsum dolor sit amet.</div>
    <div id="info">Minima maxime exercitationem cumque voluptates.</div>
    <div id="work">Facilis, voluptatem? Et, mollitia ratione.</div>
    <div id="team">Laborum fugiat doloribus tempora consectetur?</div>
    <div id="about">Sunt, earum? Eius, vitae corrupti!</div>
  </div>
</div>
      
label {
  cursor: pointer;
}

.cont div {
  display: none;
}

input:checked+label {
  color: #f40;
}

input[id='home']:checked~.cont #home {
  display: block;
}

input[id='work']:checked~.cont #work {
  display: block;
}

input[id='info']:checked~.cont #info {
  display: block;
}

input[id='team']:checked~.cont #team {
  display: block;
}

input[id='about']:checked~.cont #about {
  display: block;
}
JavaScript
操作DOM
通过更换元素的 class 或样式 style 实现
初态时，各内容隐藏
单击时，为目标叠加激活样式
因为操作DOM，所以结构可以任意
每次单击时，需要分别遍历标签页标题和内容；遍历所有标题，移除 active，然后为当前标题添加活动类 active；遍历所有内容，设置 display为 none，然后设置当前内容的 display为 block
案例实现请访问 Codepen - TabCard
item.classList.toggle('active'')      
,item{
  display: none;
}
.item.active{
  display: block;
}
