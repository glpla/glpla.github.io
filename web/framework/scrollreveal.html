@ScrollReveal

简介 Overview
说明
. 随着屏幕的滚动，逐渐显示指定内容
. 可以指定类型丰富的显示效果
步骤
1. 打开 cdnjs ，查找ScrollReveal，使用提供的在线资源
2. 选择版本，复制js对应的标签链接到项目中；如果本地展示，请复制其中的链接，在浏览器中打开，右键选择"另存为"，保存js文件
3. 进入 scrollrevealjs 官网，参照引导Guide和API，创建自己的效果
4. 实例化ScrollReveal，并指定要显示的元素，通常使用类名；多个类以逗号,区分
const sr = ScrollReveal();
sr.reveal('.class1, .class2');
5. 以JSON值对的形式，为实例和元素指定选项，设置丰富的显示效果；涉及到的时间单位是毫秒ms
实例选项
distance 距离
delay 延迟显示时间
duration 持续时间
origin 统一指定显示来向：top | right | bottom | left
reset 每次触发都显示效果
元素选项
interval 作用于多个元素时，显示的间隔
origin 单独指定显示来向：top | right | bottom | left 
const sr = ScrollReveal({
    distance: '40px',
    duration: 2000,
    delay: 400,
    reset: true
})
sr.reveal('.banner-item', {
    interval: 100
})
sr.reveal('.i0', {
    origin: 'top'
})
sr.reveal('.i4, .breath-item, .info-item', {
    origin: 'bottom',
    interval: 100
})
6. 更多使用信息，请参考官网文档
