滚动显示

@ScrollReveal
. 随着屏幕的滚动，逐渐显示指定内容
. 可以指定类型丰富的显示效果
. 进入 官网 - scrollrevealjs，使用提供的在线资源或者使用老师提供的本地资源 单击下载 scrollreveal.min.js；还可以在 cdnjs中 ，查找ScrollReveal 并使用
步骤
引入
<script src="../lib/scrollreveal.min.js"></script>
实例化 ScrollReveal - 以JSON值对的形式，为实例和元素指定选项，设置丰富的显示效果；涉及到的时间单位是毫秒ms
const sr = ScrollReveal({
    distance: '40px',
    duration: 2000,
    delay: 400,
    reset: true
})
应用到不同的元素 - 可以指定/追加新属性
sr.reveal('.banner-item', {
    interval: 100
})

sr.reveal('.i0', {
    origin: 'top'
})

也可以应用到多个类；多个类以逗号,区分

sr.reveal('.i4, .breath-item, .info-item', {
    origin: 'bottom',
    interval: 100
})
在浏览器中预览效果
选项和属性
实例选项
属性 说明
distance 距离
delay 延迟显示时间
duration 持续时间
origin 统一指定显示来向：top | right | bottom | left
reset 每次触发都显示效果
元素选项
属性 说明
interval 作用于多个元素时，显示的间隔
origin 单独指定显示来向：top | right | bottom | left
显示指定元素

先写配置再应用到实例 - 通过展开并追加属性可以为不同的元素应用不同的时间间隔

const revealOption = {
  delay: 300,
  distance: "50px",
  duration: 500,
  easing: "ease-in-out",
  origin: "bottom",
};
ScrollReveal().reveal(".swiper-info", { ...revealOption, interval: 350 });
显示完毕后执行数字动画 - 个人网站
同时使用 Anime.js库
<div class="data-item">
  <input type="text" class="num" value="234" data-val="234">
</div>
<div class="data-item">
  <input type="text" class="num" value="168" data-val="168">

ScrollReveal().reveal(".data-item", {
  afterReveal: (el) => {
    let ipt = el.querySelector('.num');
    console.log('ipt', ipt);
    anime({
      targets: ipt,
      value: [0, ipt.value],
      duration: 2000,
      round: 1,
      easing: 'easeInExpo',
    })
  }
});
