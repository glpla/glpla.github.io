样式 CSS
容器.swiper - 指定内容大小[可选]；如图片的显示，可以指定宽高或宽高比

.不建议使用背景图片 - 渲染可能有异常

.swiper {
  width: 800px;
  margin: 100px auto;
}

.swiper-slide img {
  width: 100%;
  aspect-ratio: 16/9;
  object-fit: cover;
}
导航按钮 - 使用全局变量指定导航按钮样式
--swiper-navigation-size: 36px;
--swiper-navigation-color: #fff;
分页器pagination - F12检查并修改对应的类
.swiper-pagination-bullet {
background-color: #fff;
}

.swiper-pagination-bullet-active {
background-color: #f40;
}
脚本 JavaScript
.swiper的初始化和各种选项
const swiper = new Swiper('.swiper', {
  // Optional parameters
  direction: 'vertical',
  loop: true,

  // If we need pagination
  pagination: {
    el: '.swiper-pagination',
  },

  // Navigation arrows
  navigation: {
    nextEl: '.swiper-button-next',
    prevEl: '.swiper-button-prev',
  },

  // And if we need scrollbar
  scrollbar: {
    el: '.swiper-scrollbar',
  },
});
常用选项 Option
.每个参数选项列表的第一项是默认设置
选项
类别 说明
direction 轮播方向：水平 | vertical垂直
speed 切换速度
autoplay 自动播放：false | true
grabCursor 抓取|拖拽鼠标样式：false | true
loop 是否循环：true | false
spaceBetween 幻灯片间隔px；不要为幻灯片指定margin
mousewheel 鼠标滚轮触发：false | true
slides-per-view 轮播时可以查看幻灯片的个数；默认1张
属性
分类 说明
activeIndex 当前活动幻灯片；循环模式下容易出现异常
realIndex 幻灯片实际索引；推荐使用
autoplay 自动播放；鼠标点击修改该属性，仍然会播放一次才停止
事件
分类 说明
slideChange 当前活动幻灯切换完毕
每次幻灯片切换时，为其中的内容 swiperInfo 添加 anime 动画 - 个人网站
同时使用 Anime.js库
const swiperInfo = document.querySelectorAll('.swiper-info')

const swiper = new Swiper('.swiper', {
  // ...
});

swiper.on('slideChange', function () {
  console.log('slide changed');
  anime({
    targets: swiperInfo[swiper.realIndex].children,
    opacity: [0, 1],
    duration: 400,
    easing: "linear",
    delay: anime.stagger(400, { start: 300 }),
    translateY: [anime.stagger(40, 10), 0],
  })
});
