. 为指定DOM节点创建打字效果
. 可以CDN在线使用 或 cdnjs 或 bootcdn；也可以使用相应的链接下载到本地使用
. 点击下载typed.js
步骤
1. 打开 Github 查找typed.js，目前星级13K
2. 点击进入其在github的主页，使用CDN的方式，将目标库引入项目；也可以copy URL到浏览器地址栏中打开，再右键另存为本地.js文件，最后引入到项目中
3. 根据实例创建H5
4. 在js中，创建typed的构造函数，指定打字的目标元素和打字的内容，以及其它打字的选项，如打字速度、删除速度等等
5. 在live server中查看效果

    strings: ['hello', 'world'],
    typeSpeed: 100,
    backSpeed: 100,
    backDelay: 1000,
    loop: true,
    loopCount: 3
  });
</script>
