有序列表 Ordered list
. <ol>：无序列表 ordered list；块盒 block
. <li>：列表项 list item；块盒 block
. 元素按照一定规则排列，如：从小到大
. 可以按照类型 type：数字 1[默认]、英文字母 A、a、罗马数字 I、i等编号
. 可以使用 计数器 为定制编号
. 可以嵌套使用
. 使用较少
<ol type="" start="" reversed>
    <li></li>
    //... ...
    <li></li>
</ol>
有序列表属性
类别 说明
type 1：按数字；默认排序方式
A：按大写字母、a：按小写字母
I：按大写罗马字母、i：按小写罗马字母
start 起始值；只适用于数字排序
reversed 倒序；减到0会成为负数
结构如下 - 先使用快捷编辑如 ol>li{list$}*4 生成再微调
从4开始排序
<ol start=4>
    <li>list1</li>
    <li>... ...</li>
    <li>list5</li>
</ol>
从3开始倒序排序
<ol start=3 reversed>
    <li>list1</li>
    <li>... ...</li>
    <li>list5</li>
</ol>
按照小写罗马字母排序
<ol type="i">
    <li>list1</li>
    <li>... ...</li>
    <li>list5</li>
</ol>
按照大写字母降序排序
<ol type="A" reversed>
    <li>list1</li>
    <li>... ...</li>
    <li>list5</li>
</ol>
使用计数器定制编号 - 这里使用伪元素::marker
.coustom-ol {
  counter-reset: sn;
}

.coustom-ol li {
  counter-increment: sn;
}

.coustom-ol li::marker {
  content: '0' counter(sn)' ';
}
无序列表 Unordered list
. <ul>：无序列表 unordered list；块盒 block
. <li>：列表项 list item；块盒 block
. 元素彼此之间没有顺序
. 可以嵌套使用
. 可以使用类型 type：点 disc、方块 square、圆圈 circle等不同的项目符号
. 也可以使用CSS的伪元素选择器 ::marker 指定更为丰富的标记
. 早期广泛用于 导航 ：ul→li->a；结构清晰；避免被SEO降权；为了减少层级而又不失去SEO增强，现在导航多使用nav→a实现
<ul type="">
    <li></li>
    //... ...
    <li></li>
</ul>
无序列表 type 属性
类别 说明
disc 点
square 方块
circle 圆圈
结构如下 - 先使用快捷编辑如 ul>li{list$}*4 生成若干H5结构再微调
默认使用：会破坏页面布局；为ul指定背景色，查看更直观
<ul id="marker">
  <li>list1</li>
  <li>list2</li>
  <li>list3</li>
  <li>list4</li>
  <li>list5</li>
</ul>

样式初始化

<ul>、<ol>默认都有margin和padding，会破坏页面布局；通常初始化样式：取消其默认样式，再根据设计利用CSS重新指定样式

ol,
ul {
    margin: 0;
    padding: 0;
    list-style: none;
    box-sizing: border-box;
}
