Network
网络情况，各种资源图片、样式文件等加载情况
1. stop recording：停止网络追踪
2. clear network log：清空记录
3. preserve log：保留记录，适合页面跳转时查看
4. no Throttling：不限流；可以调整网络带宽以便查看数据加载情况，如 fast 3G、slow 3G、offline
