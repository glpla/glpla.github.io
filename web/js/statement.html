分支语句
if、if-else、if-else if-else
.如果条件condition为真，执行语句statement1；否则执行语句statement2
.多分支语句请使用switch-case
if (condition) {
  statement 1;
} else {
  statement 2;
}
if (condition1) {
  statement1;
} else if (condition2){
  statement2;
}
switch-case
.多分支匹配或 相等 判定：如果给定的值value匹配到某个分支的条件值，就执行该分支
.适合特定值，不适合范围值
.case是完全匹配：数值和类型都相同才能匹配；不会隐式数据类型转换
.大概率条件分支应放在前面
.指定default分支，可以提高代码健壮性
switch (value) {
  case 条件1:
    break;
  case 条件2:
    break;
  case 条件3:
    break;
  //...
  default:
    break;
}
?:
.三目运算符
.如果条件满足，就执行语句1；否则执行语句2
.是if-else的另外一个实现
条件?语句1:语句2;
循环语句
for
.先初始化条件[只执行一次]；也可以在for外面进行
.再判定条件是否满足；如果满足就执行语句statement；如果不满足，就结束
.条件变化，重复判断并决定继续还是结束
.省略判定条件后，变为无限循环
.如果要退出某次循环，继续下一次循环，请使用continue
.如果要退出循环，请使用break，同switch语句
for (初始化;判定条件;条件变化) {
  statement; 
}
.基本使用；变量i只能作用于for循环
for (let i = 0; i < 10; i++;) {
  console.log(i);
}
.初始化放在for外面，变量i可以被其它访问；条件变化放在循环体内
let i = 0;
for (; i < 10;) {
  console.log(i);
  i++;
}
while-do
.先判定再执行
.可以和for相互转化
.特殊领域的死循环-WSN
while(1){
  //
}
do-while
.先执行1次再判定
.使用较少
forEach
.语法更简介，但是 无法 中断或停止
for-in
.for-in 语句是一种精准的迭代语句，可以用来枚举对象的属性
.建议在使用 for-in 循环之前，先检测确认该对象的值不是 null 或 undefined
.控制语句中的 var/let 操作符不是必需的。但是，为了保证使用局部变量，推荐使用
.主要应用场合：Object、JSON
let obj = { "name": "guilin", "age": 18, "marriage": true };
for (let item in obj) {
  console.log(item, obj[item]);
}
for (var propName in window) { 
    document.write(propName); 
}
