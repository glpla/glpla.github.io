按键事件

@Key
.执行顺序：keydown → keypress → keyup
.部分事件不能使用侦听，只能使用传统事件 onXXX
键盘事件类型
类别 说明
keydown 按下，不区分大小写
keyup 弹起/释放，不区分大小写
keypress 按住并保持；区分大小写；不能识别 backspace、CTRL、shift、方向键等功能键
规范已不建议使用
常用属性
类别 说明
e.key 按键名如，f、F12
e.keycode 按键编码|键值|键码；规范已不建议使用
常用键名
类别 说明
数字1-9 1-9
字母a-zA-Z a-zA-Z
回车 Enter
功能键F1-F12 F1-F12
上键 ArrowUp
下键 ArrowDown
左键 ArrowLeft
右键 ArrowRight
其它 Escape、Tab、Capslock、Shift、Control、Meta、Alt、Delete、Backspace
复合键，如shift+8是* !、 @、 #、 $、 %、 ^、 &、 *、 (、 )
[] JD搜索优化
.页面任何位置按下s，激活搜索框
let ipt = document.querySelector('input');
document.addEventListener('keyup', (e) => {
    if (e.key == 83) {
        ipt.focus()
    }
})
[] Ctrl + Enter，提交信息
如果要使用组合键，则可以利用 event.ctrlKey，event.shiftKey，event.altKey 判断是否按下了Ctrl键、Shift键以及Alt键
[] JD快递单号查询
.为方便阅读，在原搜索框上方额外显示一个字体略大的提示框。失去焦点时，隐藏提示框；获取焦点时，如果内容不为空，则显示提示框
[] 禁止F12键
document.onkeydown = function (e) {
  if (e.key == 'F12') {
    return false;
  }
}
