换肤

@Skin
CSS 样式
通过样式的切换达到换肤的效果
为根元素 html 设置自定义属性的值来应用不同的样式：白天和黑夜
:root {
  --txt-color: #303133;
  --bg-color: #ffffff;
}

[data-theme="dark"] {
  --txt-color: #e5eaf3;
  --bg-color: #131313;
}
修改为暗夜模式
document.documentElement.setAttribute('data-theme', 'dark');
或：根据当前值切换；因为没有指定 light，所以属性为空，则应用默认的变量样式
document.documentElement.setAttribute('data-theme', isDarkMode.value ? 'dark' : 'light');
自定义属性 data-
将换肤数据绑定到data-上，获取后应用到其它元素；事件处理可以封装为函数
<div class="red" data-color="#f40">red</div>
<div class="blue" data-color="#089">blue</div>
<div class="yellow" data-color="#ff0">yellow</div>
btnRed.addEventListener('click', (e) => {
  body.style.backgroundColor = e.target.dataset.color;
})
btnBlue.addEventListener('click', (e) => {
  body.style.backgroundColor = e.target.dataset.color;
})
btnYellow.addEventListener('click', (e) => {
  body.style.backgroundColor = e.target.dataset.color;
})
CSS 变量
.获取元素节点，使用其特定的样式属性，如颜色去修改变量的值，达到换肤的效果
let mixer = document.querySelectorAll('.mixer span');
let root = document.querySelector(':root');
mixer.forEach(item => {
    item.addEventListener('click', (e) => {
        console.log(e.target.style.backgroundColor);
        root.style.setProperty('--main-color', e.target.style.backgroundColor)
    })
})
