DOM综合运用

一、实验目的
了解数据交互的基本格式JSON
掌握数据本地存储LocalStorage的基本使用
掌握fetch加载数据的过程和基本使用
二、开发环境
操作系统 Window 10+
文本编辑器 Vs Code
谷歌浏览器 Chrome
三、实验内容
准备数据：使用JSON模拟远程数据，如喜欢的歌单、心灵鸡汤等等
加载数据：通过fetch加载到项目：自动加载或事件触发加载，如鼠标单击
处理数据：处理获取到的数据，更加适合页面展示
显示数据：使用表格table或表单form或其它方式展示数据
保存数据：将某些热点保存到本地
其它功能：操作数据，如数据排序、增删、改查，并将感兴趣的数据保存到本地
参考页面 数字时钟
四、实验步骤
使用Vs Code创建H5项目/文件夹
在项目中创建相应的资源文件夹和文件，如样式文件夹css和样式文件style.css、脚本文件script.js和数据文件JSON等等
准备数据：在项目特定文件夹中创建JSON数据文件并合理组织数据
创建主页文件index.html，创建HTML5结构，用来展示数据；并使用CSS3美化HTML5结构
编辑脚本文件，加载本地数据文件；通过多次log查看每个阶段数据的状态和结果
处理获取到的数据并展示到页面，根据需要调整数据组织形式或HTML5结构
操作数据：增删改查，并将操作结果存放到本地
配合live server调试正常后，撰写实验报告
项目和实验报告整体打包压缩，提交到学习通
五、实验结果分析与体会
掌握的知识点、开发技能
碰到的问题、解决的思路和方法以及未解决的问题
其它思考
六、实验要求
规范开发；独立完成；突出个人设计特点和风格
实验报告：以图片或代码形式展示并说明各部分功能、具体内容和实现细节，按要求命名，导出为PDF
实验项目：完整项目打包压缩，以学号+姓名命名，如20231010张三；压缩包格式不限
提交实验报告和实验项目到学习通
未按要求在规定时间内提交视为无效，不得分
