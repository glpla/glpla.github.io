表单综合运用

一、实验目的
了解表单的作用
了解表单提交的基本方式：GET、POST
了解使用表单采集数据和提交数据的基本方法
掌握使用<form>、<input>、<select>、<textarea>、<button>、<file>实现个人网站的登录和注册
二、开发环境
操作系统 Window 10+
文本编辑器 Vs Code
谷歌浏览器 Chrome
三、实验内容
合理使用<form>、<input>、<select>、<textarea>、<button>、<file>设计个人网站登录和注册，具体内容包括：
头像：选择图片并个性化显示
姓名：不超过8个字符
性别：男、女
个人兴趣：多选或单选
个人简介：不超过80个字符
提交：在地址栏查看采集的数据
其它功能
参考页面 一路有你
