掌握过渡动画transition的基本使用

二、开发环境
操作系统 Window 10+
文本编辑器 Vs Code
谷歌浏览器 Chrome
三、实验内容
合理使用transition、keyframe和transform设计3D魔方，具体内容包括：
魔方3D设计，包括内外2层transform
循环自转keyframe
鼠标hover上去后展开外层transition
其它功能
