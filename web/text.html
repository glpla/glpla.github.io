基本CSS
类别 说明
font-family 字体家族；通常为一个系列
中文优先；先写中文字体，再写英文字体
font-size 字体大小
font-weight 字体重量/加粗；关键字，如bold、light；推荐使用数字，如600
font 以上3个属性的复合写法
color 字体颜色
line-height 行高；通常指定1.5倍行高；PC端通常指定为字体大小+8px的行高；可用于单行文本的垂直方向居中
letter-spacing 字符间距
hyphens 对英文来说，添加连字符的方式，一般选择auto，更为紧密
text-indent 文本缩进；通常为2em
text-align 文本对齐：默认left；right | center | justify
text-align-last 末尾文本对齐方式
text-transform 文字变化，首字大写capitalize | 大写uppercase | 小写lowercase；适用于英文
text-overflow 文字溢出处理
white-space 空白字符处理，换号、不换行等
vertical-align 行内文本对齐方式：baseline | middle | top | bottom
writing-mode 文字方向，如古文的从右到左vertical-rl
text-shadow 文字阴影；可以多次叠加，以逗号,区分
text-stroke 文字描边颜色和宽度；部分浏览器不支持；建议使用 SVG 实现
确保首先使用最适合显示中文的字体，然后是英文或其他语言的字体
Arial：西文常见的无衬线字体
sans-serif：系统默认的无衬线字体
font-family: "微软雅黑", "Microsoft YaHei", Quicksand, Arial, Helvetica, sans-serif;
text-shadow: h-shadow v-shadow blur color;
-webkit-text-stroke: length color;

分栏布局 Columns
.将大块文字分为若干栏，增加阅读性
类别 说明
column-width 最小列宽，由浏览器决定分成多少列；不能使用%
column-count 列数
columns 复合属性：column-width column-count；根据宽度或列数布局
column-span 分列元素的跨度，all通栏，如标题
column-gap 列间隔；默认是16px
column-rule 分隔符样式，同border
column-fill 填充模式，默认是平均填充balance，也可以指定auto，逐列填充；通常需指定容器高度
分栏排版，请参考个人网站、青春有我
. 分2栏，并指定最小宽度
. 分栏后，应特别注意两侧的文字的对齐
. 部分内容跨栏显示
  .col-block {
    columns: 350px 2;
  }
  
  .col-block-heading {
    column-span: all;
    background-color: #e3f7ef;
    margin-bottom: 1rem;
  }

自定义字体 @fontface
.使用@font-face定义字体
.避免用户本地没有页面使用的特定字体
.字体文件应提前准备好，并推送到服务器/网站
.多个字体效果应该分别定义，字体名字保持一致
.仅在用户端浏览页面时临时下载使用；关闭浏览器后，字体被移除
.中文字体通常比较大，字体下载时，需要时间，所以页面渲染不会立即完成，影响客户体验；请谨慎使用
.字体的定义通常放在单独的css文件，需要的时候引入；注意路径
定义 - 还可以指定颜色 color、大小 font-size等属性
@font-face {
    font-family: font_name;
    font-weight: 300;
    src: url(font_url) format("TrueType")
}
引用 - 也可以引用时再指定其它属性
@import url('xxx.css');
