盒阴影

@box-shadow
. 为元素添加阴影
. 阴影由 2-4 个长度值、可选的颜色值以及可选的 inset 关键词来规定。省略长度的值是 0
. 多个阴影使用逗号,分隔
.更多阴影使用，请参考 Neumorphism
可以把阴影当作一个元素本身的孵化器
box-shadow: h-shadow v-shadow blur spread color inset;

    background-color: #e1251b;
}

    box-shadow: unset;
    transform: translateY(0px) rotateZ(45deg);
}

    transform: translateY(0px) rotateZ(-45deg);
}
