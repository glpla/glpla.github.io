布局综合运用

一、实验目的
了解文档流和层级的基本原理和作用
掌握定位的几种类型
掌握布局常见的方式
掌握使用position、float、flex、grid完成经典布局和其它布局的思路
二、开发环境
操作系统 Window 10+
文本编辑器 Vs Code
谷歌浏览器 Chrome
三、实验内容
合理使用position、float、flex和grid设计个人网站布局，具体内容包括：
整体采用网格grid布局
主导航采用粘性sticky定位，其中的导航项采用弹性盒子flex布局
侧边栏采用固定fixed定位
图文混排采用浮动float定位
正文各部分应体现出常见经典布局，如圣杯布局、左右布局、左中右布局、九宫格布局等等
各部分H5和C3关键代码以正文的内容呈现；不要截图
四、实验步骤
创建文件夹Demo，使用Vs Code打开
创建图片文件夹imgs，拷贝图片
创建样式文件夹css、初始化样式文件reset.css和主样式文件style.css
创建主页文件index.html，引入创建的初始化样式文件和主样式文件
在主页文件中创建HTML5结构
根据HTML结构，完成初始化样式文件
根据实验内容，实现各种定位
配合live server调试并修改
五、实验结果分析与体会
掌握的知识点、开发技能
碰到的问题、解决的思路和方法以及未解决的问题
其它思考
六、实验要求
规范开发；独立完成；突出个人设计特点和风格
实验报告：采用学院统一下发的格式文件，以文字说明，配以必要的效果图片或核心代码，展示并说明数据来源、实施过程、各部分功能、具体内容和实现细节
实验报告导出为PDF，按照规定要求命名
实验项目打包压缩，压缩包格式不限，命名同实验报告
实验报告和实验项目提交到学习通
未按要求在规定时间内提交视为无效，不得分
格式规范，请参考 论文规范要点
七、拓展训练与思考
用CSS的定位设计更复杂更个性化的网页
