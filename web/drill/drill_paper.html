实验目的

掌握创建文章的常用标签

实验内容
标签：<article>、<figure> <figcaption>、<table>、<footer>
选择器：伪元素选择器::before ::first-letter ::selection、伪类选择器:nth-child、组合选择器 ~ +
样式：@font-face、columns、column-span、line-height、counter-reset counter-increment、overflow、white-space、text-overflow、CSS变量
布局：浮动float
应用：图文混排、首字下沉、溢出打点
开发环境
操作系统 Window 10+
文本编辑器 Vs Code
谷歌浏览器 Chrome
