HTML属性样式
. 除了最后2个属性外，其它样式建议使用 CSS 实现
类别 说明
border 边框
width 宽度
height 高度；通常由内容自动撑开
cellspacing 单元格间距
cellpadding 单元格填充
align 水平对齐；默认左对齐
valign 垂直对齐；默认居中 middle
colspan 列合并（向右）
rowspan 行合并（向下）
CSS高频样式
width：表格宽度
border-collapse：边框是否重叠，默认不重叠；只有指定了重叠 collapse，才可以设置行 <tr> 的边框
border：表格边框/单元格边框；单元格分离 collapse 时，不能直接给 <tr> 指定边框，而应该给 <td> 指定边框
border-spacing：单元格间距；边框重叠时，无效
table-layout: 单元格内容显示方式，默认是 auto，自动适应内容；fixed 固定宽度;
empty-cells：空单元格显示与否/不在空单元格周围绘制边框。默认显示/绘制；仅仅使用于边框不重叠的情况
初始化样式
默认情况下，<th>、<td>有1个像素的 padding
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}  

table {
  width: 100%;
  border-collapse: collapse;
}
系统会默认添加表体 <tbody>；表头 <thead>、表尾 <tfoot> 和 <caption> 需要手动添加
通常只关注行 <tr>、<th>和 <td>，他们会自动放在 <tbody>中
如果需要对表头特别处理，如固定表头，建议将 <th> 显式的放在 <thead> 中
除了直接在表格上应用合并单元格属性 colspan 和 rowspan外，其它样式建议由CSS完成
表格的样式设计应聚焦在<td>
<th>的样式如背景优先级略高，部分动态样式无法覆盖
同一页面内的表格，关键列宽应该保持一致
colgroup
在表格内定义各列特性
默认情况下各列宽度会自动分配，撑开至占满表格
指定每列的宽度 - 内联样式
更多使用，请访问 MDN - <colgroup>

隐藏表头<thead>。方法很多；最好不要none掉，而应使用定位将其移走
将<tr>显示属性由table-row（类inline-block）调整为block，以便独占一行
将<td>显示属性由table-cell（类inline-block）调整为block，从而将数据由一行展示改为多行展示
设置<td>为相对定位，并保持左侧50%的padding，预留给其伪元素
为每个<td>设置不同的伪元素::before内容，和隐藏的表头内容相对应
根据需要指定行和单元格的其它样式，如边框、背景色(略)
其它布局应根据实际情况具体分析
