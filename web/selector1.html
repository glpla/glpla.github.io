伪元素选择器

@Pseudo element
. 并不是由传统的HTML标签创建出来，而是依附于已有的标签，所以叫伪元素
selector::pseudo-element {
    property: value;
}
::before ::after
. 创建一个inline元素作为第一个first子元素、最后一个last子元素
. 必须指定伪元素 content 属性，哪怕为空；可以使用普通字符、CSS实体；也可利用attr()获取元素属性；content拼接字符串不能使用+，直接在后面添加即可
. content属性不支持变量使用var()，可以通过计数器 迂回实现
. 可按照普通元素对其进行定位、布局、旋转等等操作；注意：旋转会撑开/破坏布局，请为父元素设置overflow溢出隐藏；同样需要注意的是：他们的box-sizing也要初始化为border-box
. 伪元素的样式受父级元素的影响；但是伪元素不是元素的直接孩子，无法使用直接后代选择器>选择
. 应用非常非常非常广泛
::first-letter ::first-line
. block块元素 的首字母、首行
. 设置的样式有限；常用于首字下沉
. 是内联元素inline，无法转化为块元素block
. ::first-letter优先级高于::first-line
::selection
. 元素被选择的部分；默认是蓝底白字
::placeholder
. 占位符样式；多用于表单元素input
. 设置的样式有限
::marker
. 定义标记类型
. 适用于display为list-item的元素，如<li>、<summary>
. 使用content属性指定内容
. 设置的样式有限
. 因为标记会破坏布局，基本很少使用
. 更多案例，请参考列表 List - 自定义编号示例
[] 样式初始化
*, 
*::before, 
*::after {
    display: block;
    box-sizing: border-box;
}
//普通文本
li::before {
  content: '+';
}
//CSS实体
li::before {
  content: '\21E6';
}

//图片；大图片调整大小应转为行块盒并指定溢出隐藏
//建议使用留空的方式，采用背景图片处理
li::before {
  content: url(./imgs/avatar.jpg);
  display: inline-block;/* MUST */
  overflow: hidden;/* MUST */
}

//获取标签属性；可以是自定义属性data-
li::before {
content: attr(data-ind);
}

//留空，做其他装饰性设计
li::before {
  content: '';
}

//使用计数器；可以拼接普通字符或CSS实体
li::before {
content: counter(sn) '\21e8 ';
}
