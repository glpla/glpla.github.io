简介 overview
说明
. inline
. Scalable Vector Graphics - 可缩放矢量图形
. 使用 XML 描述 2D 图形；是万维网联盟的标准
. SVG 基于 XML：每个元素都可用附加 JavaScript 事件处理器
. 每个绘制的 SVG 图形均被视为对象。如果 SVG 对象的属性发生变化，浏览器能够自动重绘
特点
. 可以使用SVG属性；也可以使用内联样式或外部CSS样式
步骤 steps
1. 准备画布
2. 绘制内容
笔刷 stroke
stroke：颜色
stroke-opacity：透明颜色
stroke-width：宽度
stroke-dashoffset：笔刷起点绘制偏移；
stroke-dasharray：笔刷点划线段大小分配/占空比：实线段、空线段；1个参数时，表示50%；当占空比发送变化时，可以利用这个特性，可以制作描边、加载等特效
stroke-dasharray：length | percentage
stroke-linecap：笔刷端点类型，子弹形、圆形、方形
stroke-linecap：butt | round | square;
fill：颜色填充；默认黑色；如果不需要，请显式指定为none
fill-opacity：透明填充
SVG vs Canvas
Canvas 是HTML5 提供的新标签元素，而 SVG 存在的历史要比 Canvas 久远，已经有十几年
Canvas 绘制出来的图形或传入的图片都依赖分辨率，能够以 .png 和 .jpg格式保存存储图像，是位图|标量图；SVG绘制的是矢量图
Canvas 不支持事件处理器，SVG支持事件处理器
Canvas多用于统计用的图表、大型的网络游戏；SVG：地图、动态的小图标 icon
基本使用 Usage
1. circle
. 位置cx、cy；半径r
. 默认中心为(0, 0)
[] circle
<svg class="svg-basic">
    <circle cx=150 cy=150 r="150" fill="#f40"></circle>
</svg>
2. ellipse
. 位置cx、cy；xy方向上的半径rx、ry
[] ellipse
<svg class="svg-basic">
    <ellipse cx=150 cy=150 rx="100" ry="80" fill="#f40"></ellipse>
</svg>
3. rect
. 位置cx、cy；宽高width、height；圆角rx、ry
. rx、ry为圆角，为宽高的1/2时，矩形会变为椭圆或圆
[] rect
<svg class="svg-basic">
    <rect x=40 y=40 width=220 height=220 rx=0 ry=0 fill=#f40></rect>
</svg>
4. line
. 两个点(x1,y1)、(x2,y2)和笔触颜色stroke和宽度stroke-width
. stroke、stroke-width使用CSS指定；省略该属性，line不可见
. 可以配合stroke-dasharray和stroke-dashoffset绘制更丰富的线条
[] 基本线条
<svg class="svg-basic">
    <line class="svg-line" x1=0 y1=0 x2=300 y2=300></line>
</svg>
.svg-line {
    stroke: #f40;
    stroke-width: 2px;
}
[] 点划线
. 改变数字，查看点划线起点的偏移情况
stroke-dashoffset: 
.svg-line-dash {
    stroke: #f40;
    stroke-width: 4px;
    stroke-dasharray: 10 20;
}
5. polygon
. 利用点的集合points和笔触颜色stroke和宽度stroke-width或填充fill绘制
. 填充fill默认黑色；如不需要，可以设置为none
[] polygon
<svg class="svg-basic">
    <polygon class="svg-polygon" points="0,0 50,50 50,100"></polygon>
</svg>
.svg-polygon {
    stroke: #f40;
    stroke-width: 2px;
    fill: none;
}
6. text
. SVG 允许您像图形一样修改文本并应用渐变、图案、剪切路径、蒙版或过滤器
. 默认情况下，text使用以下默认属性：颜色：黑色；字体大小：16px
. 水平对齐 text-anchor: start | middle | end；默认start
. 垂直对齐 dominant-baseline: hanging | ideographic |alphabetic | middle | central
. x,y：表示定位点；以该点为参考，放置文本，比如水平居中、垂直居中
[] 静态描边文字
. SVG容器为320*120，定位点为160*60；当水平居中middle垂直居中central时，文字位于容器正中间
. 在画布外，增加一个<div>放置背景图片
. 给text增加文字阴影[2重]
<svg class="svg-txt">
    <text class="title-static" x="160" y="60" text-anchor="middle" dominant-baseline="central">遇见最好的自己</text>
</svg>
.svg-txt {
    width: 320px;
    height: 120px;
    margin: 0 auto;
}

.title-static {
    height: 120px;
    line-height: 120px;
    font-size: 40px;
    font-family: 'yuwei';
    text-shadow: 0 0 10px #ff0, 0 0 10px #ff0;
    fill: none;
    stroke: #fff;
    stroke-width: 1;
}
[] 动态描边文字
. SVG容器为320*120，定位点为160*60；当使用水平垂直居中属性时，文字位于容器正中间
. 通过stroke-dashoffset的变化实现动态描边
<svg class="svg-txt">
    <text class="title" x="160" y="60" text-anchor="middle" dominant-baseline="central">路在前方人在路上</text>
</svg>

路径 path
. SVG 基本形状中最强大的一个。你可以用它创建线条，曲线，弧形等等
. path 元素的形状是通过属性 d 定义的，属性d的值是一个“命令 + 参数”的序列
. 每一个命令都用一个关键字母来表示
. 大写字母，表示采用绝对定位；另一种是用小写字母，表示采用相对定位
基本命令
分类 说明
M 移动
L 绘制线
H 简写命令；绘制水平线
V 简写命令；绘制垂直线
A 曲线
C 贝塞尔曲线；至少3个点
Q quadratic Bézier curve 二次贝塞尔曲线
两组坐标 x1 y1 x y 分别是控制点和终点，控制点的作用是确定起点和终点的曲线斜率
Z 闭合路径
绘制弧线
A rx ry x-axis-rotation large-arc-flag sweep-flag x y
rx：轴半径
ry：轴半径
弧形的旋转角度：x-axis-rotation
large-arc-flag：决定弧线是大于还是小于 180 度，0 表示小角度弧，1 表示大角度弧
sweep-flag：表示弧线的方向，0 表示从起点到终点沿逆时针画弧，1 表示从起点到终点沿顺时针画弧
sweep-flag：弧线方向
x y：弧形的终点
<svg width="400" height="200">
  <path d="M 0 -200 A 3 1 0 1 0 400 0 V 200  A 3 1 0 1 0 0 300 Z" fill="#f40" />
</svg>
