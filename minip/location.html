常用属性
属性 说明
longitude 经度；必填
latitude 纬度；必填
show-location 显示带有方向的当前定位点
marker 在地图上显示标记的位置；结合地图服务API - 地点搜索实现
属性
latitude：标记经度；必填
longitude：标记维度；必填
iconPath：标记图片路径；必填
id：标记id；number；注意是数字类型；每个标记的id应该不同
width：标记宽度
height：标记高度
label：标签；在标记底部显示信息；对象类型
callout：冒泡；默认情况下，单击标记，在上面显示信息；对象类型；手机端才能看到效果
wx.getLocation(Object object)
获取当前的地理位置、速度
只针对部分限定类目的小程序开放
需要开通权限：[开发] → [开发管理] → [接口设置]中自助开通
需要在 app.json 中进行声明
更多信息，请访问 官方文档
wx.createMapContext(string mapId, Object this)
根据地图组件map的mapId创建其上下文 MapContext 对象
MapContext 通过 id 跟一个 map 组件绑定，操作对应的 map 组件；常用方法有：
1. moveToLocation(Object object)
将地图中心移置当前定位点；地图组件需设置 show-location 为true
2. getCenterLocation(Object object)
获取当前地图中心的经纬度；返回 gcj02 坐标系

页面.js - 获取位置信息
data: {
    longitude: 0,
    latitude: 0,
},

onLoad: function (options) {
    wx.getLocation({
      type: 'gcj02',
      success: res => {
        this.setData({
          longitude: res.longitude,
          latitude: res.latitude
        })
      }
    })
}

封装酒店搜索函数getHotel(lat, lng)：借助系统函数区域改变regionChange()获取的结束点end位置来更新当前区域的酒店信息

移动地图时，更新函数

注意：this的使用

getHotel( latitude,longitude) {
    let _this = this;
    qqmapsdk.search({
        keyword: '酒店',
        location: {
        latitude: latitude,
        longitude: longitude
        },
        success: (res) => {
        let markers = [];
        for (let index = 0; index < res.data.length; index++) {
            markers.push({
            id: res.data.length,
            longitude: res.data[index].location.lng,
            latitude: res.data[index].location.lat,
            iconPath: '../../utils/imgs/marker.png',
            title: res.data[index].title,
            address: res.data[index].address,
            width: 20,
            height: 20
            })
        }
        _this.setData({
            markers
        })
        },
        fail: (res) => {
        console.log(res);
        },
        complete: (res) => {}
    })
}
regionChange(e) {
    if (e.type == 'end') {
        this.getHotel(e.detail.centerLocation.latitude, e.detail.centerLocation.longitude)
    }
}
