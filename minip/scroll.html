滚动

Scroll
实现页面或部分内容的滚动
wx.pageScrollTo(Object object)
将页面滚动到目标位置，支持 选择器 和 滚动距离 两种方式定位
更多信息，请访问 官方文档 - pageScrollTo
pageScrollTo常用属性
参数 说明
scrollTop 滚动到页面的目标位置；单位 px
duration 滚动动画时长；单位 ms；默认300
selector 选择器；仅仅支持ID、类、后代、并列等基本选择器
offsetTop 偏移距离，需要和 selector 参数搭配使用，可以滚动到 selector 加偏移距离的位置，单位 px
返回顶部 - 将某个元素固定定位在页面底部，绑定事件；API使用如下
使用距离
wx.pageScrollTo({
    scrollTop: 0,
    duration: 300,
    success: res => console.log(res),
    fail: err => console.log(err),
    complete: () => console.log('scroll done')
})
使用选择器 - 顶部容器类为container
wx.pageScrollTo({
    selector:'.container',
    duration: 300,
    success: res => console.log(res),
    fail: err => console.log(err),
    complete: () => console.log('scroll done')
})
返回到页面某个元素 - 不太适合使用距离，因为需要知道元素的位置
