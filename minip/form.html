表单

<form>
<form>和其中的大多数表单元素，基本同HTML5保持一致；具体效果视系统而定
表单数据提交时，必须指定name；value可以由事件获取
常用组件
表单组件繁多，常见的分类如下表
<form>

表单容器

提交时，可以在事件对象e.detail中获取表格各元素的值；要求每个表单指定name属性

bindsubmit 携带数据提交；默认
bindreset 表单重置时会触发 reset 事件
label

用for来改进指定了id的表单组件的可用性

<checkbox value="撸码" id="code" /><label for="code">撸码</label>

使用lable包裹表单元素时，应略去for属性

<label>
    <checkbox color="#3498db" value="撸码" />撸码
</label>
<input>

输入框

属性 说明
value 输入框初始内容；字符串；必填
type
输入框类型
text：文本键盘；默认
number：数字键盘
digit：带小数点的数字键盘
idcard：身份证键盘
password 是否是密码类型；布尔；默认假
placeholder 输入框为空时占位符；字符串
maxlength 最大输入长度；数字
focus 获取焦点
confirm-type
设置键盘右下角按钮的文字，仅适用text类型
send：发送
search：搜索
next：下一个
go：前往
done：完成；默认
bindinput 键盘输入时触发
bindfocus 输入框聚焦时触发
bindblur 输入框失去焦点时触发
bindconfirm 点击完成按钮时触发
<radio> <radio-group>

单选按钮；单选按钮组

属性 说明
value 单选按钮的值
checked 当前是否选中
color 单选按钮的颜色；同accent-color
bindchange 选中项发生改变时触发；从事件对象e.detail获取当前选中的值

实际应用中，利用给定的数据和当前值是否相等来决定哪个是选中状态

<radio-group bindchange="getGender" name="ugender">
    <radio value="男" checked="{{userInfo.ugender=='男'}}" color="#3498db" />男
    <radio value="女" checked="{{userInfo.ugender=='女'}}" color="#3498db" />女
</radio-group>
<checkbox> <checkbox-group>

多选按钮；多选按钮组

属性 说明
value 多选按钮的值
checked 当前是否选中
color 多选按钮的颜色；同accent-color
bindchange 选中项发生改变时触发；从事件对象e.detail获取当前选中的值
<textarea>

多行输入；默认大小300*150；block

属性 说明
value 输入框初始内容；字符串；必填
placeholder 输入框为空时占位符；字符串
maxlength 最大输入长度；数字
focus 获取焦点
confirm-type
设置键盘右下角按钮的文字，仅适用text类型
send：发送
search：搜索
next：下一个
go：前往
done：完成；默认
bindinput 键盘输入时触发
bindfocus 输入框聚焦时触发
bindblur 输入框失去焦点时触发
bindconfirm 点击完成按钮时触发
bindlinechange 输入框行数变化时调用
<button>

功能强大的按钮；建议只设置背景和字体颜色

除了默认的tap事件外，按钮其它事件大多和相应的开发能力关联

属性 说明
loading 是否带 loading 图标
form-type
按钮类型；点击触发 form 的submit/reset 事件
submit：提交表单
reset：重置表单
open-type
微信开放能力
chooseAvatar：获取用户头像；从bindchooseavatar回调函数的detail中返回头像临时地址
contact：客服；需要绑定客服事件bindcontact
getPhoneNumber：获取用户手机(企业号)
getUserInfo：获取用户信息(接口被收回)
<picker>

从底部弹起的滚动选择器，如：时间选择器、日期选择器、地区选择器

除了下述通用的属性外，不同的mode，picker拥有不同的属性；如时间类的都有开始start、结束end等，详情请查看 官方文档

属性 说明
mode
模式
selector：普通选择器
multiSelector：多列选择器
time：时间选择器
date：日期选择器
region：省市区选择器
bindcancel 取消选择时触发
<slider>

滑动选择器

属性 说明
min 最小值
max 最大值
step 步长
show-value 是否显示值；默认不显示
backgroundColor 背景条的颜色
activeColor 已选择的颜色
block-size 滑块的大小，取值范围为 12 - 28
block-color 滑块颜色
bindchange 拖动后触发
bindchanging 拖动过程中触发
<switch>

开关选择器

属性 说明
checked 是否选中
color 已选择的颜色
bindchange 拖动后触发
组件还有一个共同属性：disabled
