使用时间格式化依赖 Moment.js
安装依赖

在小程序根目录，右键 → 打开内建终端|小黑屋

npm install moment --save
构建NPM

菜单栏 → 工具 → 构建 npm

引入包

在逻辑中引入

const moment = require('moment')
使用包

默认格式

console.log(moment().format());

指定格式 - 使用 durations

changeTime(t) {
    let time = moment.duration(t, 'seconds')
    let h = time.hours()
    let m = time.minutes()
    let s = time.seconds()
    return moment({
        h,
        m,
        s
    }).format('HH:mm:ss')
}
