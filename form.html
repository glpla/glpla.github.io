form

表单属性
action 表单提交后端时的操作
method 表单提交的方式；默认是GET
新增类型
datalist：以下拉列表的形式，为input等文本输入元素提供一个输入可选项；使用list属性和对应文本元素的id绑定在一起
fieldset、legend：分组表单元素，并指定分组的标题；分组的好处是操作整个分组，如：使用disabled属性禁止整个分组
input新类型
email、tel、url：验证型，请配合pattern和JS完善验证需求
rang、number、color、date、time、month、week：取值型
input新属性
autocomplete：提示历史输入，方便自动完成，一般配合datalist
autofocus：自动获取焦点
placeholder：内容占位符，用以展示提示信息
required：必填
pattern：利用正则表达式提供验证功能，如：pattern="^[a-zA-Z]\w{4,9}$"
其他属性
checked：input的默认选中
selected：select的默认选中
maxlength：最大字符数
name/value：值对，以区分不同的数据
id：配合<label>使用，增强用户体验

input

作用：收集信息给后端服务器server；通常配合<label>和各种属性增强用户体验

1 单行文本输入框 type=text；默认类型

    <label for="user">user</label>
    <input type="text" name="user" id="user" required maxlength="8" placeholder="your name">

2 密码输入框 type=password

    <input type="password" name="pw" id="" required maxlength="8" placeholder="your password">

3 单选按钮 type=radio

    <input type="radio" name="gender" value="male" id="" checked>male
    <input type="radio" name="gender" value="female" id="">female

4 多选框 type=checkbox

    <input type="checkbox" name="hobby" value="reading" id="" checked>reading
    <input type="checkbox" name="hobby" value="coding" id="">coding
    <input type="checkbox" name="hobby" value="game" id="">game
    <input type="checkbox" name="hobby" value="movie" id="">movie

5 按钮 type=submit

    <input type="submit" value="submit">
    //也可以直接使用<button>
    <button>submit</button>

文件选择file

文件是<input>的一种，类型为file；默认选择单个文件，可以指定multiple为多选；accept是文件类型，类型之间使用,分割，也可以使用通配符*

    <input type="file" name="avatar" id="" multiple accept="*">

文件操作fileReader

处理选中的文件。根据内容，有3个方法：

1 readAsText()：处理.txt/.HTML/.js等文本文件

2 readAsDataURL()：处理图片

3 readAsArrayBuffer()：处理字节流

读取本地文本文件
提示：请选择文本文件。格式应为为UTF-8
    txt.addEventListener('change', function () {
      let file = this.files[0]
      let fr = new FileReader()
      fr.readAsText(file)
      fr.onload = function (e) {
        console.log(e.target.result);
        txtCont.innerHTML = e.target.result
      }
    })

  let excelCont = document.querySelector('#excel-cont')
  let excel = document.querySelector('#excel')
  excel.addEventListener('change', function () {
    let fr = new FileReader()
    fr.readAsArrayBuffer(this.files[0])
    fr.addEventListener('load', function () {
      let res = this.result;
      let workbook = XLSX.read(res, { type: 'array' });
      let sheetName = workbook.SheetNames[0];
      let sheet = workbook.Sheets[sheetName];
      console.log(sheet);
      excelCont.innerHTML = XLSX.utils.sheet_to_html(sheet);
    })
  })
  //请查看sheet内容，定制excel文档

思考

  • 如何知道单个文件的信息：name、size和type
  • 如何选择多个文件：multiple属性
  • 如何知道选择文件的个数：length；单个文件时，length为1
  • 文件类型都有哪些
  • 如何指定选择文件的类型：accept属性
  • 如何自定义UI而不使用系统默认的丑陋的样式：none掉input，利用额外元素如button的click来触发input的click（change）
  • 如何拖拉获取文件信息：定义拖拉区，并指定drop事件。注意：系统默认会打开拖入的文件，应取消dragenter和dragover的默认动作
  • 如果查看文件内容：FileReader
