Pointer

case 1：修饰的是int，不允许修改int的值

    const int a = 10;
    a = 11;//error: assignment of read-only variable 'a'

case 2：修饰的是*p，不允许修改*p的值，也就是不允许使用指针变量修改数据；但是指针可以指向其它数据

    int a = 10;
    const int *p;
    *p = 10;
    //error: assignment of read-only location '*p'：
    //case1和case2实际上是一种情况

case 3：修饰的是p，指针指向的地址不能变，只能在声明的时候给指针赋值，后面就不可乱指

    int a = 10;
    int *const p;
    p = &a;//error: assignment of read-only variable 'p'
    //int *const p = &a;需要声明即赋值
