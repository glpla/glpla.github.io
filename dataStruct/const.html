常量的分类

常量是指程序执行过程中，值不发生改变；分为：直接常量和符号常量；
1. 直接常量也称为字面量，可以直接拿来使用，如12、-1.414、'a'，"hello world"； 字符常量：以ASCII码来表示，本质上还是一个int数字，如符号常量'a'实际上是整型97，其它：
A-Z：65-
a-z：97-
0-9：48
2. 符号常量：宏，原样替换，注意使用()来区分运算的优先级；符号常量必须先定义define再使用，如#define unit 54414

常量的使用

修饰哪个就限制哪个；后面跟什么就限制什么

case 1：修饰的是int，不允许修改int的值

    const int a = 10;
    a = 11;//error: assignment of read-only variable 'a'

case 2：修饰的是*p，不允许修改*p的值，也就是不允许使用指针变量修改数据；但是指针可以指向其它数据

    int a = 10;
    const int *p;
    *p = 10;
    //error: assignment of read-only location '*p'：
    //case1和case2实际上是一种情况

case 3：修饰的是p，指针指向的地址不能变，只能在声明的时候给指针赋值，后面就不可乱指，指向NULL也不可以

    int a = 10;
    int *const p;
    p = &a;//error: assignment of read-only variable 'p'
    //int *const p = &a;需要声明即赋值
