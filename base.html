CSS
CSS文件实例
    @charset "utf-8";//[必须是第一行；如果出现中文，必须指定；通常不需要指定；因为不推荐使用中文]
    @import "./style.css";//[其它样式文件]

    :root {//[全局变量声明]
        --txt-wran-color: #c81623;
        --nav-bg-color: #9f103a;
        --w-width: 1000px;
    }

    //样式：由选择器和样式声明组成。样式声明采用值-对的形式
    img {
        width: 100%;
        object-fit: cover;
    }
位置
1.外部样式：以.CSS文件的形式，通过<link>引入到HTML的头部<head>；推荐且必须
                <link rel="stylesheet" href="css/reset.css">
2.内部样式：在HTML头部<head>的<style>中书写CSS样式
                <style type="text/css">    
                  selector {
                  attribute:value;
                }
3.内联样式：在H5元素中写在style属性中。通常配合JS来动态改变页面样式
                <h1 style="text-align: center;">login</h1>
特性
继承性：子元素会继承父元素的某些样式；不是所有的样式都可以被继承
重叠性：元素同一个属性的样式，后面的样式会覆盖前面的样式
优先级：一般来说，同一个元素，优先级高的样式会覆盖优先级低的；外部样式<内部样式<内联样式；同时还要结合不同选择器的权值来决定最终使用的是哪个样式；如果样式没有生效，请检查是否优先级不够

根据需要，还衍生出很多组合使用，见综合实例
其它选择器

并列/群组选择器：以,分隔

    ul,li {
      margin: 0;
      padding: 0;
    }

后代选择器：以空格分隔，从所有后代中选择符合条件的元素

    ul li{//选中所有的<li>
      text-transform: capitalize;
    }

直接后代选择器：以>分隔，从儿子中选择符合条件的元素；注意某些属性如：color、fong-size的继承性

    .nav>li{
      margin: 0 1rem;
      }

弟弟选择器：以~分隔，从所有弟弟中选择符合条件的元素

    .item1~li {//选中item2、item3、item4
      color: #078;
    }

相邻小弟弟选择器：以+分隔，如果相邻的弟弟符合条件，则选中该元素

    .item3+.item4 {//选中contact节点元素
      color: #4f0;
    }

结构伪类选择器：以:分隔

    .sub-nav li:nth-child(2) {//选中CSS节点元素
      text-decoration: underline;
    }

字体图标
以阿里字体图标为例：
1.注册用户
2.创建项目；和网站无关，仅仅是为了管理字体图标；
3.查找字体图标；先逐个添加到购物车，最后再添加到项目；
4.可以在线使用；也可以下载到本地使用；阿里字体图标支持Unicode、font[推荐：更直观] class和symbol三种方式使用；
说明
.当做普通字体使用：设置颜色、大小等字体属性；
.一般使用<i>或<span>元素展示阿里字体图标；
.支持彩色图标；
.每次往项目中添加、删除字体时，都需要重新更新一下CSS文件；
在线字体
字体以CSS的形式提供，只需要把对应的字体文件CSSS引入即可；
    //at.alicdn.com/t/font_2979815_i60gj7m30tf.css
自定义字体
避免用户本地没有页面使用的特定字体；仅在用户端浏览页面时临时下载使用；关闭浏览器后，字体被移除；字体下载时，需要时间，所以页面渲染不会立即完成，影响客户体验；谨慎使用；
1.准备字体：推荐使用谷歌字体[支持在线字体]；也可以使用其它平台提供的字体文件；下载到当前网站某个目录；
2.声明
    @font-face {
        font-family: "YuWei";
        src: url('../fonts/YuWei/YuWeiShuFaXingShuJianTi.ttf') format("TrueType")//通常需指定多个字体文件
    }
3.使用
    .core {
        font-family: "YuWei";
        //...
    }

HTML新标签

结构标签

<article>：完整的文章结构；通常包括以下新增标签

<section>：类似于<div>，但是HTML5建议应带上标题<header>；如果不需要标题，使用<div>即可

<header>：头部信息，通常用在<body>;<article>和<section>头部

<nav>：导航，使用<ul>实现；顶部、侧边、分页导航

<aside>

<footer>：底部信息，通常位于<body>;<article>底部，包括：版权声明;友情链接;备案信息等

其他标签

<small>：配合<footer>，提供版权声明;备案信息;免责声明等

<progress>：动态进度条

<meter>：静态进度条

<figure>：建议取代<img>
