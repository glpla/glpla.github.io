简介

行内元素inline

可以实现页面内部、页面之间之前的跳转

可以包含其它tag，但不可以嵌套<a>

在head区加上<base target="">指定默认打开方式。

语法


    <a href="" target=""></a>

属性

href链接的目标URL；可以是HTML页面、图片、视频、文件等
target打开目标URL的位置；默认是_self，当前页面打开

锚点跳转

同一个页面内部，不同部分之间的跳转；便于查看和阅读

    //目标元素：指定id
    <div id="p3">锚点跳转内容区</div>
    //<a>：指定href为目标元素的id名；注意应带上id符号#
    <a href="#p3">锚点跳转</a>

页面跳转

页面之间的跳转；根据页面是否属于同一个网站，又可以分开站内跳转和站外跳转

    //站内跳转：同一个网站内部页面之间的跳转
    <a href="greeting.html">greeting</a>
    //站外跳转：跳转到其它网站的页面；可以不指定网络协议，由系统自行判断
    <a href="https://www.baidu.com">baidu</a>
    <a href="//www.baidu.com">baidu</a>
