文章排版

(必须显式的指定行高)

俗话说人靠衣装，美靠靓装，一篇文章要想将其完美呈现还是需要有好的排版才行。一个优秀的排版可以使别人在阅读的时候，更加的赏心悦目，以此来吸引更多的人群去阅览。

一、内容选择

1.字体

微软的雅黑字体是我们最常使用的字体，但要注意的是同样大小的微软雅黑字体看上去的显示会比其他字体大一点，因此，当我们在设置字体大小的时候，12 px-16 px大小的文本是最合适的，一般使用14 px大小的为多。

把握好字体之间的行间距，一般1.5到1.75的行间距最适合。对于字数较多的文章，通常可以选择使用左对齐，字数较少的就可以选择居中对齐。

2.标点符号

标点符号不要乱用；中文输入法与英文输入法的标点符号表示是不同的，我们要将其分清开来。中文与英文之间要用空格。中文数字得到序号要使用顿号，比如一、二、三。

3.图片

配图要有调性；首先要确定你所写文章的定位调性，然后去配上与文章整体风格比较相符的图片。

配图尺寸大小：配图的尺寸要协调一致，横图竖图的尺寸大小一样，这样可以提升整体的视觉效果。

图片不加水印：你从网站下载的那么有水印的图片当然是不能使用的，它会降低你文章的整体美感。

图片太小太模糊：图片像素太小太过模糊的图片就不要放上去了，配图上连内容都看不清，那么逼格一下就会没有。

图片过大：我们所放入的每一张图片都不要超过100KB,PNG格式的一般都比JPEG格式的容量要小，为了保持清晰度而又不增加图片的容量，我们可以选择使用PNG格式。如果超过200KB的图片就会太大，读者在阅读时就会加载慢，这样就会严重的影响到用户的体验感。

二、样式选择

1.水平对齐

中文的行文习惯是首行缩进2个字符：text-indent:2em;

为了减少行末的锯齿，应采用两端对齐：text-align: justify;

2.垂直对齐

如果内容由多种标签提供，如<p>、<h1>、<h2>等，则会出现垂直方向上移位的情况

这是因为元素占据的高度不一致导致的。最简单的方法就是指定各个标签的行高line-height，同时保证行高是最小元素的整数倍即可。

也可以通过指定每个元素的padding和margin，以保证所占空间是最小元素空间的整数倍。

三、特殊效果

1首字下沉

首字下沉需要通过伪元素::first-letter来实现。其实现原理非常简单，就是把首字font-size值设置较大，设置合适的边距，最后使用float来实现。

.cap::first-letter {
  float: left;
  font-size: 3em;
  color: #f40;
  margin: 1rem 1rem 1rem 0;
}

因为用到了浮动，所以必须要清除浮动。新的属性initial-letter还未广泛普及，可以使用浏览器前缀尝试一下。

2.自动滚动

受布局尺寸的影响，应当根据内容的多少，自动出现水平方向或垂直方向的滚动条。可以使用的属性有：overflow: auto; overflow-x: auto; overflow-y: auto;

还有很多其它样式和特效的设置，请同学们自行查阅相关文档。

最后，要保证文章当中无错别字，当我们排版完成后，还要再去检查一下，看看整篇文章中是否有错别字，给用户一个严谨严肃的形象。
