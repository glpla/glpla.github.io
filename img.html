<img>

属性
src：图片来源
title：hover时，要显示的额外信息
alt：图片异常时的提示信息
样式
width
height
object-fit
vertical-align
  使用模式1：直接使用图片 

图片指定宽度，其高度自适应。图片高度太大，布局被撑开；图片高度太小，布局被压缩

  .img {
    width: 360px;
  }
  使用模式2：使用容器包裹图片 

容器指定宽高并溢出隐藏；图片指定宽度100%，其高度自适应。

容器高度太大，留有空白；容器高度太小，图片展示不完全

  .img-box {
    width: 360px;
    height: 360px;
    background-color: #ccc;
    overflow: hidden;
  }
  
  .img-box .img {
    width: 100%;
  }
  使用模式3：直接使用图片 

图片指定宽度、高度。同时指定object-fit

  .img {
    width: 360px;
    height: 360px;
    object-fit: cover;
  }
  图片对齐方式 
vertical-align: baseline;
vertical-align: middle;
vertical-align: top;
vertical-align: bottom;
