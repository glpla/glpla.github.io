模板语法

Mustache
也称数据绑定
Vue 使用一种基于 HTML 的模板语法，使能够声明式地将其组件实例的数据绑定到呈现的 DOM 上，减少 DOM 操作
可以使用简单的js表达式实现数据的组合使用；复杂的逻辑请使用计算属性或方法获取
根据数据是否是响应式，数据绑定分：静态数据绑定和 响应式数据 绑定；本节使用静态数据绑定
目的：解决数据和元素节点的内容和属性的关联
可以使用模板字符串``，便于直接使用变量
内容绑定 {{}}
文本插值表达式
使用双大括号 {{}}，会将数据解释为 纯文本 ，解决 数据和元素内容 的关联
支持简单的数据处理；建议使用 计算属性 完成
简单数据类型绑定和复杂数据类型绑定
let msg = 'hi,there'
let age = 18
let obj = {
    id: 100,
    name: 'gl'
}
let url = 'http://www.baidu.com'
<div>msg: {{ msg }}</div>
<div>handle with msg: {{ msg.substring(3) }}</div>
<div>age: {{ age }}</div>
<div>handle with age: {{ age + 2 }}</div>
<div>id: {{ obj.id }} - name: {{ obj.name }}</div>
属性绑定 v-bind
使用 v-bind 指令绑定HTML的属性Attribute，解决 数据和元素属性 的关联
可以简写为冒号 :
常见的元素属性有：href、id、title、src、class、style以及布尔属性如hidden、disabled、autoplay、loop、controls、checked、selected等
类和样式的使用请参考 类和样式
一般属性
<a :href="url" target="_blank">link</a>
<div :id='uid'>my id</div>
布尔属性

静态使用 - 指定的任何数据包括空串，都会渲染为真；可用简单的指定属性，不做任何赋值，实现假的操作

<button disabled="true">submit</button>
<button disabled="false">submit</button>
<button disabled=" ">submit</button>
<button disabled="123">submit</button>
<button disabled>submit</button>

动态绑定使用 - 可用指定关键字true或false、布尔变量或结果为布尔值的表达式

<button :disabled="true">submit</button>
<button :disabled="false">submit</button>
           
<button :disabled="isEnable">submit</button>

<button :disabled="age > 20">submit</button>
图片的使用

静态使用 - 可以使用本地图片或在线图片，如pixabay、unsplash

<img src="../assets/imgs/0.jpg" alt="">

不可以直接绑定图片url地址，vue会将其解析为一个字符串，而非地址；解决方案和更多使用，请访问图片

let src = '../assets/imgs/0.jpg'
<img :src="src" alt="">
CSS函数属性绑定

background-image等使用了url()、linear-gradient()等等CSS函数的属性

静态使用 - 仅可以引用public中的资源

<div style="background-image: url('/imgs/lg.jpg');">hi,there</div>

其它内容绑定
v-text：渲染普通文本
v-html：渲染HTML元素；但是不建议：在网站上动态渲染任意HTML非常危险
let txt = 'hi, there.'
let html = '<p>warning</p>'
<div v-text='txt'></div>
<div v-html='html'></div>
