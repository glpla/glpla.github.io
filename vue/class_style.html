class vs style
Overview
CSS class 列表和内联样式 style 都是 attribute，使用 v-bind 将它们和动态的 字符串 绑定
v-bind:可以简写为冒号:
除了字符串外，表达式的值也可以是 对象 或 数组
绑定类和静态添加的类不会冲突，最终会叠加在一起进行渲染
注意双引号和单引号的配套使用
使用vue devtool调试
内联样式 :style
复合属性使用小驼峰方式
指定样式值 - color
<div :style="{ backgroundColor: color }">content</div>
拼接样式值 - 数值和单位
<div :style="{ lineHeight: h + 'px' }">content</div>
拼接样式值 - url和地址
<div :style="{ backgroundImage: 'url(' + imgUrl + ')' }">content</div>
多样式使用逗号,分隔，渲染时变为分号;
<div :style="{ backgroundColor: color, lineHeight: h + 'px' }">content</div>
类 :class
. 根据条件决定是否使用某个类；条件称为开关；类称为开关类
. 开关为布尔数据；为真时，使用开关类；为假时，不使用开关类；也称内联字面量开关类
<div :class="{ active: isActive }">hi,there.</div>
. 开关为表达式
<div :class="{ active: ind==1 }">hi,there.</div>
. 开关为对象；可以是一个开关，如{ active: isActive }或多个开关{ active: isActive, done: isDone}
<div :class="isOn">hi,there.</div>
const isOn = reactive({
    active: false,
    done: true
})
. 使用计算属性
const isActive = ref(false)
const isDone = ref(true)
const isOn = computed(() => ({
    active: isActive.value,
    done: isDone.value
}))
. 使用三元表达式有条件地渲染某个 class
<div :class="isActive ? 'bold' : 'normal'">hi,there.</div>
导航/标签页
单击导航列表，对应的导航项高亮
每个导航项对应一个数字；数字变换时，对应的导航项应用活动类active
静态类和动态类可以分开单独写；也可以以数组的形式写在一起
<nav>
    <a href="" class="item" :class="flag == 0 ? 'active' : ''">item1</a>
    <a href="" class="item" :class="flag == 1 ? 'active' : ''">item2</a>
    <a href="" class="item" :class="flag == 2 ? 'active' : ''">item3</a>
    <a href="" :class="['item', flag == 3 ? 'active' : '']">item4</a>
</nav>
作用域 scope
样式部分中，使用scoped属性指定样式仅仅作用于当前组件。但是会增加额外的开销
渲染时，每个组件分配一个属性
内部样式是通过组件的属性进行选择
解决的方法：为组件的根节点指定一个特有的类。如.nav。推荐以组件名字命名这个类；因为组件的功能各不相同，所以很容易保证组件类的唯一性
