插槽

@ slot
插槽 slot 用以 拓展 组件的功能：在基本功能的基础上，通过插槽插入内容，从而实现功能扩展
使用插槽 slot 可以像使用 HTML 元素一样向组件中传递内容；可以是静态的，也可以是动态的
可以指定插槽默认内容 - 默认插槽
可以声明多个插槽。为了区分，多个插槽应该指定 name 属性 - 具名插槽
基本过程
1. 子组件创建过程中，预留一个位置，声明slot，将其作为一个占位符，父组件传递进来的内容就会渲染在这里
仅预留位置
<template>
    <div>
        //内容
        <slot />
    </div>
</template>
默认插槽 - 只有一个插槽的情况；可以指定默认内容
<template>
    <div>
        //内容
        <slot>插槽默认内容</slot>
    </div>
</template>
具名插槽 - 多个插槽；使用属性 name 区分；同样可以指定默认内容
<template>
    <div>
        //内容
        <slot name='header'></slot>
        <slot name='footer'></slot>
    </div>
</template>
2. 父组件
使用内容填充插槽；可以是普通文本，也可以是HTML节点
<template>
    <AlertBox> Something bad happened. </AlertBox>
</template>
使用默认插槽
<template>
    <AlertBox> </AlertBox>
</template>
使用具名插槽 - 使用一个使用 v-slot 指令标识的 <template> 元素
v-slot可以简写为 #
<template>
    <AlertBox>  
        <template #header>
            <h1>Here might be a page title</h1>
        </template>
        <template #footer>
            <h1>Here might be a page title</h1>
        </template>
    </AlertBox>
</template>
