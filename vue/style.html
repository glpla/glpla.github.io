样式

style
类 class 和内联样式 style 都是元素的属性 attribute，使用 v-bind 将它们和动态的 字符串 绑定
v-bind:可以简写为冒号:
除了字符串外，表达式的值也可以是 对象 或 数组
绑定类和静态添加的类不会冲突，最终会叠加在一起进行渲染
复合属性使用小驼峰方式
注意双引号和单引号的配套使用
使用vue devtool调试
内联样式 :style
指定样式值 - color
<div :style="{ backgroundColor: color }">content</div>
拼接样式值 - 数值和单位
<div :style="{ lineHeight: h + 'px' }">content</div>
拼接样式值 - url和地址
<div :style="{ backgroundImage: 'url(' + imgUrl + ')' }">content</div>
多样式使用逗号,分隔，渲染时变为分号;
<div :style="{ backgroundColor: color, lineHeight: h + 'px' }">content</div>
