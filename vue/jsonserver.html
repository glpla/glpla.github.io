json-server

JSONServer
说明
NPM地址：json-server
快速搭建HTTP服务
跟随文档操作即可
更多使用细节，请访问从入门到膨胀
方法
GET - 数据查询；配合id查询特定记录
POST - 新增数据；会自动增加id
PUT|覆盖 和 PATCH|更新 - 修改数据
DELETE - 删除数据
基本使用
安装 - json-server
npm install json-server
创建数据文件 - students.json；包括用户id，姓名和星数；初始应为空，通过添加和修改逐步增加
{
  "digital": [
    {
      "uid": 20220110,
      "uname": "gl",
      "rank": 0
    }
  ],
  "software": [
    {
      "uid": 20220210,
      "uname": "gl",
      "rank": 0
    }
  ]
}
监听数据文件 - students.json

默认端口3000

json-server students.json

指定端口 - 和Vue项目端口保持一致，避免浏览器跨域

json-server students.json --port=5173
请求数据 get - 挂载完毕
onMounted(() => {
  axios.get('http://127.0.0.1:5173/software')
    .then(res => {
      //赋值
    })
})

可以封装函数

onMounted(() => {
  loadData()
})
func loadData(){
  axios.get('http://127.0.0.1:5173/software')
    .then(res => {
      //赋值
  })
}
数据渲染 - 判断数据是否为空
<div class="wrap">
  <template v-if="students.length">
    <div class="item" v-for="(item, ind) in students" :key="item.id">
      <div>{{ ind + 1 }}</div>
      <div>{{ item.uid }}</div>
      <div>{{ item.uname }}</div>
      <div>
        <span class="iconfont icon-iconfontshoucang" v-for="star in item.rank"></span>
      </div>
      <button @click="mod(item.id, item.rank)">增加</button>
    </div>
  </template>
  <template v-else>数据为空</template>
</div>
增加数据 post - 提交用户id，姓名和星数；完毕后重新拉取数据
function add() {
  axios.post('http://127.0.0.1:5173/software', {
    uid: uid.value,
    uname: uname.value,
    rank: rank.value
  })
    .then(res => {
      loadData()
    })
}
修改数据 patch - 根据id或uid

由事件对象获取结构中绑定的数据

function mod(id, rank) {
  axios.patch('http://127.0.0.1:5173/software/' + id, {
    rank: rank + 1
  })
    .then(res => {
      //更新完毕后，创新加载数据
      loadData()
    })
}
删除数据 delete - 根据id或uid
function del(id) {
  axios.delete('http://127.0.0.1:5173/software/' + id)
    .then(res => {
      loadData()
    })
}
