作用

建议前后端加强沟通
认真制定并落实开发规范、接口API

安装
打开Vs Code的命令行终端 CTRL + ~，或使用Window系统 Win + R，运行CMD
编译
1.手动编译 - tsc file.ts
2.自动编译 - 根据配置文件自动将.ts转换为.js并输出到指定目录
初始化为tsconfig.json：tsc --init
修改严格模式为：strict: false
修改编译输出路径：outDir: "./js"
"终端Terminal" → "运行任务 Run Task" → "显示所有任务 Show All Tasks" → "tsc watch..."
