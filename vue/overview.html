模板语法
说明
Vue 使用一种基于 HTML 的模板语法，使我们能够声明式地将其组件实例的数据绑定到呈现的 DOM 上
将组件实例的数据绑定到呈现的 DOM；减少 DOM 操作；也称数据绑定
可以使用简单的js表达式实现数据的组合使用；复杂的逻辑请使用计算属性或方法获取
每个数据绑定仅支持单一的表达式
目的：解决数据和元素节点的内容和属性的关联
文本插值
解决数据和元素内容的关联
使用双大括号 {{}}，会将数据解释为纯文本
let msg = 'hi'
<span>Message: {{ msg }}</span>
<span data-v-e17ea971="">Message: hi</span>
特别的，使用 v-html 指令可以使用HTML元素
不建议使用；在网站上动态渲染任意HTML非常危险
let html = '<p>warning</p>'
<div v-html='html'></div>
<div data-v-e17ea971="">
    <p>warning</p>
</div>
属性绑定 v-bind
解决数据和元素属性的关联
使用 v-bind 指令绑定HTML的属性Attribute
可以简写为 :
let url = 'https://glpla.github.io'
<a :href="url" target="_blank">link</a>
<a href="https://glpla.github.io" target="_blank" data-v-e17ea971="">link</a>
特别的，当属性为布尔型，如disable，只要数据不为假false，其他任何数据包括空串，都会渲染为真
let isEnable = false
<button :disabled="isEnable">submit</button>
<button data-v-e17ea971="">submit</button>
