双向绑定

@defineModel()
通过编译宏 defineModel() 的形式实现通信|参数传递|数据双向传递 - recommended, Vue 3.4 +
早期使用 v-model 实现双向绑定 - two-way binding
更多信息，请访问 Component v-model
子组件
const model = defineModel()

function update() {
  model.value++
}
<div>Parent bound v-model is: {{ model }}</div>
<button @click="update">Increment</button>
父组件
<Child v-model="countModel" />
