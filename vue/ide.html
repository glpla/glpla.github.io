项目构建
说明
使用项目创建指令，根据提示操作即可
$ npm create vite@latest
详细步骤
1. 在项目存放目录，打开CMD，输入以下指令，创建项目并指定项目名
2. 选择Vue框架 - 使用方向键选择，回车确认
3. 定制Vue框架 - 只配置基本的功能，typescript、语法检查等功能暂且不选
4. 根据提示进入项目目录、安装依赖并运行项目
按住CTRL + 单击项目链接，即可打开Vue + Vite的欢迎界面
5. 恭喜，您已经打开了Vue开发的大门
项目创建完毕后，后续开发使用Vs Code完成依赖安装和项目预览
