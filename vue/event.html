事件

Event
事件绑定 v-on
监听 DOM 事件；可以简写为 @
<a v-on:click="doSomething"> ... </a>

<a @click="doSomething"> ... </a>
事件类型
包括：鼠标事件、键盘事件
常见事件类型
类型 说明
click 鼠标单击
dbclick 鼠标双击
keyup 某键松开
keydown 某键按下
keyenter 回车按下
mouseup 鼠标松开
mousedown 鼠标按下
mousemove 鼠标移动
mouseover 鼠标悬停
事件修饰符
限制事件特性
常见事件修饰符
类型 说明
.stop 停止父级事件的传播/冒泡
.once 只触发一次
.prevent 阻止默认事件
.native 使用原生事件；多用于组件
定义在组件上的事件默认都是自定义事件，即：系统认为是子组件$emit出来的事件；如需使用原生时间，应指定事件修饰符
按键事件修饰符
类型 说明
.enter 配合enter键
.alt 配合alt键
.alt.enter 配合alt键enter键
... ...
事件对象 $event
使用普通形参的时候，可以使用关键字$event获取事件对象
