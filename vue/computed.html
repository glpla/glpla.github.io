计算属性

Computed
说明
数据绑定时，可以使用简单的表达式实现数据的2次处理
根据age判断DOM的渲染
<div>{{ age >= 18 ? 'adault' : 'teenage' }} </div>
逻辑复杂时，可以使用计算属性实现
优点
计算属性是响应式
数据响应时，不会刷新页面
需要引入 computed 包
计算属性仅会在其响应式依赖更新时才重新计算
计算属性有返回值
相比之下，方法调用总是会在重渲染发生时再次执行函数
使用value指定属性值
import { ref, computed } from 'vue';
let age = ref(18)
const ageRes = computed(() => {
    return age.value >= 18 ? 'adault' : 'teenage'
})
