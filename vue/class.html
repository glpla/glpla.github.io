class
类 class 和内联样式 style 都是元素的属性 attribute，使用 v-bind 将它们和动态的 字符串 绑定
v-bind:可以简写为冒号:
除了字符串外，表达式的值也可以是 对象 或 数组
绑定类和静态添加的类不会冲突，最终会叠加在一起进行渲染
复合属性使用小驼峰方式
注意双引号和单引号的配套使用
使用vue devtool调试
类 :class
. 除了直接在结构中指定类-静态类外，还可以根据条件决定使用动态类
. 条件也称开关或开关变量，相应的动态类称为开关类
. 开关的本质是布尔变量，所以也可以是逻辑表达式或比较表达式的结果
. 可以分开使用静态类和动态类；也可以使用数组形式集中使用
绑定类

用不用这个类？ - isActive为真，使用bold类；为假，不使用；还可以指定一个对象，对象中对多个类的使用进行约束

<div :class="{ 'bold': isActive }">hi,there</div>
let isActive = false

用哪个类？ - 用三元表达式实现2选1；isActive为真，使用bold类；否则使用underline类

<div :class="isActive ? 'bold' : 'underline'">hi,there</div>

用哪个类？ - 通过更改类名实现更多类的选择；tips为bold，使用bold类；tips为underline，使用underline类；还可以使用数组指定多个类

<div :class="tips">hi,there</div>
let tips = 'bold'
.bold {
    font-weight: 600;
}
.underline{
    text-decoration: underline;
}

类名需要使用引号包裹，否则会被当作变量；注意单引号和双引号的匹配

绑定内联样式style

可以使用小驼峰camelCase，也支持短划线 kebab-cased；推荐使用小驼峰camelCase，因为静态使用和动态绑定写法一致

静态使用样式 - 简单属性和复合属性

<div style="color: #f40;">hi,there</div>
<div style="fontSize: 20px">hi,there</div>
<div style="background-color: #f40;">hi,there</div>

绑定属性值 - 属性使用什么值，采用 值对key-value 的形式；值key只能是小驼峰camelCase

<div :style="{ color: txtColor }">hi,there</div>
<div :style="{ fontSize: small + 'px' }">hi,there</div>
<div :style="{ backgroundColor: bgColor }">hi,there</div>
let txtColor = '#f40'
let small = 20
let bgColor = '#089'

类的绑定使用较多，不要纠结于内联样式的使用

样式和类是元素的属性绑定，属性由属性名和属性值组成；属性值需要用引号括起来，所以 :class=""
开关变量控制开关类：用还是不用，是值对形式，是一个对象，JSON规范中，key要使用引号，所以 :class="{'active':isActive}"
三元表达式的结果是一个字符串，所以直接写就可以 :class="flag == 0 ? 'active' : 'bold'"
数组：可以存放数字和字符串等多种数据类型；类一定是字符串，所以类应该用括号括起来，而变量不用，所以静态类和动态类 :class="['active',{'active':isActive}]"
单引号和双引号注意匹配使用
//开关为布尔变量；一个或多个
<div :class="{ 'active': isActive }">hi,there.</div>
<div :class="{ 'active': isActive, 'done': isDone }">hi,there.</div>

//开关为比较表达式
<div :class="{ 'active': ind==1 }">hi,there.</div>

//使用三元表达式
<div :class="{isActive ? 'bold' : 'normal' }">hi,there.</div>

//同时使用静态类和动态类 - 分离形式 - 推荐
<div class="item" :class="{flag == 0 ? 'active' : 'normal'}">item1</div>

//同时使用静态类和动态类 - 数组形式
<div :class="['item', { 'active': isActive }]">item4</div>

//直接绑定对象，由对象的一个开关或多个开关决定动态类；应用较少
<div :class="isOn">hi,there.</div>

const isOn = reactive({
    active: false,
    done: true
})
//使用计算属性
const isActive = ref(false)
const isDone = ref(true)
const isOn = computed(() => ({
    active: isActive.value,
    done: isDone.value
}))

作用域 scope
样式部分中，使用scoped属性指定样式仅仅作用于当前组件。但是会增加额外的开销：渲染时，每个组件分配一个属性
内部样式是通过组件的属性进行选择
推荐：为组件的根节点指定一个特有的类。如.nav。推荐以组件名字命名这个类；因为组件的功能各不相同，所以很容易保证组件类的唯一性
