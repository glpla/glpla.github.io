class
类 class 和内联样式 style 都是元素的属性 attribute，使用 v-bind 将它们和动态的 字符串 绑定
v-bind:可以简写为冒号:
除了字符串外，表达式的值也可以是 对象 或 数组
绑定类和静态添加的类不会冲突，最终会叠加在一起进行渲染
类 :class
用不用这个类？ - 对象 方式；isActive为真，使用bold类；为假，不使用；还可以指定一个对象，对象中对多个类的使用进行约束

值对key-value是一个对象，JSON规范中，key要使用引号

<div :class="{ 'bold': isActive }">hi,there</div>
<div :class="{ 'active': isActive, 'done': isDone }">hi,there.</div>
<div :class="{ 'active': ind==1 }">hi,there.</div>
用哪个类？ - 字符串方式；通过更改类名实现更多类的选择；tips为bold，使用bold类；tips为underline，使用underline类

简单字符串 - 字符串即类名

<div :class="tips">hi,there</div>

对象 - 由对象的一个开关或多个开关决定动态类

<div :class="isOn">hi,there.</div>
const isOn = reactive({
    active: false,
    done: true
})
三元表达式实现2选1；isActive为真，使用bold类；否则使用underline类

三元表达式的结果是一个字符串，所以直接写就可以，不需要{}

<div :class="isActive ? 'bold' : 'underline'">hi,there</div>
同时使用静态类和动态类 - 数组方式

数组可以存放数字和字符串等多种数据类型；类一定是字符串，所以类应该用括号括起来，而变量不用

数组中可以使用字符串、对象和三元表达式指定类

<div :class="['item', { 'active': isActive }]">item4</div>
<div :class="['item', isActive ? 'bold' : 'underline']">hi,there</div>
同时使用静态类和动态类 - 分离方式
<div class="item" :class="{flag == 0 ? 'active' : 'normal'}">item1</div>
类名需要使用引号包裹，否则会被当作绑定的变量
注意单引号和双引号的匹配

作用域 scope
样式部分中，使用scoped属性指定样式仅仅作用于当前组件的元素。但是会增加额外的开销：渲染时，每个组件分配一个属性
内部样式是通过组件的属性进行选择
推荐：为组件的根节点指定一个特有的类。如.nav。推荐以组件名字命名这个类；因为组件的功能各不相同，所以很容易保证组件类的唯一性
