单文件组件

@SFC
SFC - Single File Component，每个文件都是一个组件
文件名采用大驼峰，后缀名为 .vue；如 HomeView.vue
将一个 Vue 组件的模板、逻辑与样式封装在单个文件；遵循：结构 - 样式 - 逻辑 "三分离" 原则
基于 SFC 开发的应用程序，称为单页面应用 SPA - Single Page Application；所有的业务都在一个页面内完成
更多信息，请访问 Single-File Components
<template>
    //结构
</template>

<script>
    //逻辑
</script>

<style>
    //样式
</style>
