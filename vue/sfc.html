SFC
单文件组件 - Single File Component，每个文件都是一个组件
文件名采用大驼峰，后缀名为.vue；如HomeView.vue
将一个 Vue 组件的模板、逻辑与样式封装在单个文件；遵循：结构 - 样式 - 逻辑"三分离"原则
<template>
    //结构
</template>

<script>
    //逻辑
</script>

<style>
    //样式
</style>
结构和样式同vue2相比，没有什么变化
逻辑除了选项式API，还提供了组合式API
1. 选项式 Options API
<script>
    export default {
        data() {
            return {
            }
        },
        methods: {
        },
        mounted() {
        }
    }
</script>
2. 组合式 Composition API - 按需导入，并采用setup语法糖；我们采用组合式 Composition API
<script setup>
    
</script>
