全局前置守卫
使用 router.beforeEach 注册一个全局前置守卫
接收3个参数：
1. from：离开的路由
2. to：进入的路由
3. next：路由拦截，如：未登录不允许访问，并引导|重定向到指定页面
正常访问
router.beforeEach((to, from, next) => {
    console.log('router from', from.path, 'to ', to.path);
    next()
})
不允许访问
router.beforeEach((to, from, next) => {
    console.log('router from', from.path, 'to ', to.path);
    if(to.path=='/home')
        next(false)
})
更多路由控制，应该由后端服务器实现，前端不可靠
