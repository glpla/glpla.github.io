三、实验内容
服务器端：在前续静态资源服务器搭建的基础上，准备json数据并处理，相应用户特别路由的请求
客户端：从资源服务器拉取数据并渲染
四、实验步骤
准备json数据文件；位置、格式和内容自定
在服务器脚本中，增加文件系统fs，解析Jason数据文件
const fs = require('fs')

const file = fs.readFileSync('./public/data/food.json')
const jsonFile = JSON.parse(file)
增加路由，响应用户的get请求
app.get('/food', (req, res) => {
  res.json(jsonFile)
})
启动服务，利用fehelper或postman调试服务器和接口是否正常
创建Vue项目，修改项目，仅保留主入口组件App.vue
逻辑设计：请求资源服务器，拉取数据
结构设计：根据拉取的数据综合运用响应式数据、条件渲染和列表渲染完成商品列表页
