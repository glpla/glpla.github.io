自动侦听器

WatchEffect
立即运行一个函数，同时响应式地追踪其依赖，并在依赖更改时重新执行
概述
更多信息，请访问watchEffect()
需引入包 watchEffect
应用：页面加载时，立即加载当前对应的数据
watch 仅在侦听的数据源确实改变时才会触发回调；执行0-N次
watchEffect 会立即执行回调函数，也称自动侦听器，等同于使用了立即配置项immediate；执行1-N次
watch 必须显式的指定某个侦听的数据源|变量
watchEffect 会自动侦听数据源|变量
watch 控制更为精细；deep、immediate
分析下面代码的功能
watchEffect(() => {
    str.value = msg.value.split('').reverse().join('')
})
使用watchEffect自动侦听多个数据
停止侦听
要手动停止一个侦听器，请调用 watch 或 watchEffect 返回的函数
停止侦听
const stopWatch = watch(num, (newv, oldv) => {
    console.log('num is watched', newv, oldv);
    if(newv > 5){
        stopWatch()
    }
})
