准备数据

选择或创建一个文件夹如public作为各种资源的主目录；利用各种子文件夹存放不同的资源数据，如data存放json、imgs存放图片、music存放音乐等等

1.1 json数据：创建json数据文件rank.json；这里以暴露2个接口为例；一个是info；一个是cont；这里包括了图片的src，可以给出完整路径，也可以仅仅给出文件名；数据加载后再2次处理

注意：id为字符串类型

{
  "info":{},
  "cont":[
    {
      "id":"1001",
      "name":"gl",
      "age":18,
      "rank":1,
      "src":"1.jpg"
    },
    {
      "id":"1002",
      "name":"cq",
      "age":20,
      "rank":2,
      "src":"2.jpg"
    }
  ]
}
