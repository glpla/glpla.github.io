响应式

Responsive
ref
创建基本数据类型、复合数据类型的响应式数据
ref() 将传入参数的值包装为一个带 .value 属性的 ref 对象
ref 的 .value 属性是响应式
1. 引入包 ref - 下述各例略
import { ref } from 'vue'
2. 声明响应式变量 - 执行函数、传入参数、变量接收
const count = ref(0)
3.1 结构使用 - 直接使用，vue会自动获取其value
<div>{{count}}</div>
3.2 逻辑使用 - 赋值时需额外指定value属性，其它情况直接使用，如监听
console.log(count) //{value:0}
console.log(count.value) //0
reactive
只能创建复合数据类型的响应式数据，如对象、数组
不用指定value属性
需引入包 reactive - 下述各例略
import { reactive } from 'vue'
不能声明简单数据类型
const count = reactive(10) //value cannot be made reactive: 10
不能重新赋值，否则就失去响应式
const obj = reactive({
    id: 10,
    name: 'gz'
})

//在某个事件函数中重新赋值，则objRe就会丢失响应式，结构中使用的变量不再响应数据变化
const doRea = () => {
    objRe = {
      id: 20,
      name: 'gz'
    }
}
只能单个属性赋值
const doRea = () => {
    objRe.id = 20
}
经过函数ref、reactive处理后，普通的数据或对象就变成了响应式
实际开发中，更多使用ref
