@ Component
SFC - Single File Component，单文件组件
vue中每个组件都是以SFC的形式存在，后缀名是.vue
包括：结构<template>、逻辑<script>和样式<style>，其中，结构<template>是必须的
每个组件可以被其它组件导入|引入并多次使用
适合模块化开发和团队协作
基本过程
创建 - 新建子组件 Demo.vue，指定后缀名为.vue
<script setup>
    //逻辑
</script>
    
<template>
    //结构
</template>

<style>
    //样式
</style>
引入：父组件 Super.vue在<script>引入子组件，注意 ：不能省略后缀名.vue；路径关系
<script setup>
    import Demo from './Demo.vue'
</script>
使用：父组件 Super.vue在结构<template>中使用子组件；可以多次使用
<template>
    <h1>Here is a child component!</h1>
    <Demo />
    <Demo></Demo>
</template>
