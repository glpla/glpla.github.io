过渡动画 Transition

说明

元素从一个状态过渡到另外一个状态

通常配合元素的hover实现

过渡自动双向执行。只要状态翻转，过渡就会执行

复合属性语法

transition: property duration timing-function delay;

单独属性语法

1.transition-property：哪个属性参与过渡，通常指定为all

2.transition-duration：过渡持续时间，单位是秒；过渡时间必须指定(其它参数可选)，且必须是第一个时间参数；第二个时间参数是延迟时间；UI界面的过渡通常不超过0.4s，以免拖沓

3.transition-timing-function：时间函数，指过渡的效果/方式，默认是匀速线性linear

4.transition-delay：过渡延迟时间；单位是秒；默认0秒；延迟必须为第二个时间参数；第一个时间参数是过渡时间

注意

使用单独属性语法可以指定多个过渡属性，必须是一一对应

一般参与过渡的属性都是数值型的；不是每个属性都可以参与过渡

步进过渡动画

由时间函数 steps(number_of_steps，direction)指定的一种特殊过渡效果

number_of_steps：数字，过渡分几个阶段完成

direction：start和end，默认是end，指每个阶段过渡结束后才渲染；start则是过渡开始前就渲染

steps()

矩形的宽度从50px，经过5个阶段，过渡到300px

steps(5,end)
steps(5,start)

为了避免状态翻转的自动反向过渡，可以分别指定过渡的时间。如下

    .step-demo-end {
      width: 50px;
      height: 50px;
      background-color: #f40;
      transition: 0s;
    }
    
    .step-demo-end:hover {
      width: 300px;
      transition: 5s steps(5);
    }
