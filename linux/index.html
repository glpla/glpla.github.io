5.关机 Shutdown
shutdown [-c][-r] time
说明
-c：取消
-r：重新启动
time：now，立即关机；如果不指定时间和参数，1分钟后关机
如果是服务器，请谨慎关机！！！
[案例] 重启：shutdown -r now
[案例] 在等待关机的过程中，取消关机：shutdown -c
[案例] 从现在开始5分钟后关机：shutdown +5
[案例] 20:30关机：shutdown 20:30
命令 Command
1.命令格式
格式：command [-options] [parameter]
说明：区分大小写!!!选项以-开头；参数不需要-；[]表示可选的；选项一定在参数前面；
2.常见命令
pwd：Print Work Directory，输出当前所在目录；使用最多；没有之一
ls：list[同dir]，显示当前或指定目录文件；
-a：隐藏文件
-l：列表显示详细信息
-R：recursion递归显示
-h：更加直观的显示文件大小
cd：Change Directory，更改工作目录；
cd：切换到用户主目录
cd ~：切换到用户主目录；也可以切换到用户主目录下某个文件夹，如cd ~/Desktop/
cd ..：切换到上级目录
cd .：切换到当前目录
cd -：在最近两次工作目录之间切换
touch：新建文件；鸿蒙系统的碰一碰；
文件不存在，则创建；文件存在，修改最近访问时间；
mkdir：make directory，创建工作目录；不允许同名；
-p：嵌套创建，如：
[案例] mkdir -p a/b/c/d
rm：remove，删除文件；不能恢复哦！！！
-r：删除非空目录；通常情况下，非空目录不能删除；
-f：强制删除，不给任何提示，即使不存在也会执行删除操作[简单粗暴型]；
clear：清屏；不能简写为cls哦；
[案例] 根据学校、学院、年级、专业、班级的架构，在桌面Desktop创建对应的文件夹，并在班级文件夹中创建一个以学号+姓名命名的txt文件。验收后，逐级返回Home；
3.查看命令帮助
格式：command --help
格式：man command
说明：man是manual/手册；
4.通配符
操作[选择、删除等等]文件时，使用通配符更加方便、精准、高效；
* 任意个数的字符
? 任意一个字符
[abc] 匹配abc中任意一个字符
[a-b] -表示范围；匹配a到b中任意一个字符
[案例] ls *3.txt：显示以3结束的所有txt文件；
[案例] ls *3*：显示以3结束的所有文件，类型不限；
[案例] ls ?2?.txt：文件名中间是2，长度为3的txt文件；
[案例] ls ?[123]?.txt：文件名中间是1或2或3，长度为3的txt文件；
[案例] ls ?[1-9]*.txt：文件名第2位是1-9某个数字的所有txt文件；
5.进阶命令
tree：树型显示当前或某个工作目录；
-d：只显示目录
如果不支持，请根据提示安装：sudo apt install tree
cp src des：copy，拷贝文件/目录；通常会带上相应的目录；如果复制时不想改文件名字，只需指定目标目录；
-i：information，如果有同名文件覆盖，则给出提示
-r：递归复制目录以及其中的内容，相当于整体复制文件夹
[案例] 拷贝到上级目录下的b目录，并更改名字：cp 123.txt ../b/321.txt
[案例] 拷贝到上级目录下的b目录，不更改名字：cp 123.txt ../b
[案例] 拷贝当前目录下目录b的内容到a目录里的c目录下；如果c目录不存在，则创建；如果c存在，则拷贝到c目录下：cp -r ./b a/c
mv src des：move，移动文件；通常会带上相应的目录；移动目录时，不需要额外的选项，如-r；可以在移动的过程中更改文件名；
用法1：移动文件
[案例] 移动b目录的112.txt到当前目录：mv b/112.txt .
用法2：移动文件的同时更改文件名
[案例] 移动b目录的122.txt到当前目录并改名221.txt：mv b/122.txt 221.txt
[案例] 为避免同名覆盖，可以带上-i参数，给出提示信息：mv -i 112.txt 114.txt
用法3：更改目录
[案例] 更改b目录为bb目录：mv b bb

find 目录 -name ""
查找文件：在指定目录下查找符合条件的文件；如果不指定目录，就在当前目录下查找；可以使用通配符；
ln -s
软链接；快捷方式；s：shortcut；todo
tar -cvf 包名.tar 目录|文件
tar -xvf 包名.tar
分别是打包和解包；打包过程不负责压缩；-cvf和-xvf固定搭配使用；打包多个文件，请使用空格分隔；
tar -zcvf 包名.tar.gz 目录|文件
tar -zxvf 包名.tar 目录
指定-z选项可以实现压缩或解压缩；执行过程：先压缩再打包；先解压缩再解包；
gzip

文件 File
1.概述
ls -l命令显示文件信息如下表所示。如果不是或具备相应的权限，就置-；
.目录
.拥有者权限：home下的用户
.组权限：要使用系统，必须是合法用户；每个用户拥有的权限不同；为了高效管理用户[权限]，通常将用户划分到不同的组
.其它用户权限
目录 拥有者权限 组权限 其它用户权限
d r w x r w x r w x
4 2 1 4 2 1 4 2 1
说明：r：read，可读；w：write，可写；x：excute，可执行[访问，如cd]修改权限，请使用chmod命令；
.硬链接数：访问的方式有多少中
.拥有者名称：修改所有者，请使用chown命令；
.组名称：修改组，请使用chgrp命令；
.大小
.时间
.名称：普通文件：白色；可执行文件：绿色；文件夹：蓝色
2.权限
修改文件权限；-表示减少权限；+表示增加权限；权限变更时，文件图标有相应的变化；
chmod -R +/-rwx 文件名|文件目录
也可以使用权限对应的权进行修改[推荐]
chmod -R num|num|num 文件名|文件目录
[案例] 修改文件权限：chmod -rw 01.py
[案例] 修改文件权限：chmod +r 01.py
[案例] 修改目录权限：chmod -x demo
[案例] 修改目录权限：chmod -rw demo
修改文件所有者
chown 用户名 文件名|文件目录
3.文件
[案例] 创建隐藏文件：touch .123.txt并显示；
cat：concatenate，查看文件内容、创建文件、追加内容等；
-b：仅非空行显示行号
-n：显示行号，包括空行
[案例] 显示文件121.txt内容：cat 121.txt
more：分页显示结果
[案例] 分屏显示文件121.txt内容，配合以下热键控制显示效果：more 121.txt
space 每按一次显示下一屏
enter 每按一次显示下一行
b 每按一次回滚一页
f 每按一次前滚一页
q 退出
grep：以高亮的形式显示文件中含有特定内容的行；相当于查找/搜索
-n：匹配行并显示行号
-v：排除内容，即：不包括指定内容的行，相当于取反搜索
-i：忽略大小写
^a：以a开头的行
a$：以a结束的行
[案例] 搜索文件中包含as的行：grep as 112.txt
[案例] 搜索文件中的不包括as的行并显示行号：grep -vn as 112.txt
[案例] 搜索文件中以as开头的行并显示行号：grep -n ^as 112.txt
[案例] 搜索文件中以as结束的行并显示行号：grep -n as$ 112.txt
echo：一般情况下，在终端输出内容；
>/>>：重定向，重新指定结果去向，即：将原本在终端显示的结果追加或覆盖到指定文件中，用于保存结果；如果文件不存在，则创建；可以不用空格分隔；相比touch创建文件，更加方便、直接；
[案例] 覆盖文件内容：echo hi,there >221.txt
[案例] 追加文件内容：echo hi,there >>221.txt
[案例] 将查看的结果放到文件中：ls -lh >112.txt
[案例] 将查看的结果追加到文件中：tree >>112.txt
|：管道，将一个命令的输出，作为另一个命令的输入，相当于多级过滤；通常配合more和grep命令使用；
[案例] 在显示主目录的结果中查找包含cc的行：ls -lha ~ | grep cc
[案例] 分屏显示主目录结果：ls -lha ~ | more
用户 User
cnplaman@cnplaman:~$
username @ hostname : workdirectory
1.用户类型
1.1 超级用户
root；权限最大，用于管理和维护系统；安全起见，不推荐使用该用户登陆系统；
1.2 标准用户
home家里的用户，由系统自动创建；
1.3 sudo用户
su是substitute user，替代用户，允许普通用户替代超级用户root执行某些操作；使用时必须先输入密码；只有5分钟的有效期，过期则需要重新输入密码；
2.组的管理
系统的组信息存放于etc/group目录下
1 groupadd 添加组
2 groupdel 删除组
3 cat /dtc/group 确认/查看组
4 chgrp -R 组名 文件|目录 修改组
[案例] 在当前用户桌面创建一个demo文件夹；新增一个组dev；将demo文件夹的组修改为dev；
3.用户管理
3.1 用户查看
id [username] 查看用户UID和GID；如果不指定用户，则查看当前用户
who 查看当前登陆的所有用户[0表示本机用户，其它为用户对应IP地址]
whoami 查看当前登陆的用户 
    cnplaman@cnplaman:~$ who
    cnplaman tty2         2022-06-13 15:27 (tty2)
    cnplaman@cnplaman:~$ whoami
    cnplaman
    cnplaman@cnplaman:~$ id
    uid=1000(cnplaman) gid=1000(cnplaman) groups=1000(cnplaman),4(adm),24(cdrom),27(sudo),30(dip),46(plugdev),122(lpadmin),134(lxd),135(sambashare)
    cnplaman@cnplaman:~$
3.2 用户切换
su username：su表示switch user；参数-表示同时切换至用户家目录；切换时需要提供密码；不带用户名，则切换到root[不推荐]
exit：返回原先用户

    cnplaman@cnplaman:~$ su - root
    Password: 
    root@cnplaman:~# exit
    logout
3.3 设置root用户密码
sudo passwd root：需俩次确认
