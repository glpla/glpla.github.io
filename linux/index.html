5.关机 Shutdown
shutdown [-c][-r] time
说明
-c：取消
-r：重新启动
time：now，立即关机；如果不指定时间和参数，1分钟后关机
如果是服务器，请谨慎关机！！！
[示例]重启：shutdown -r now
[示例]在等待关机的过程中，取消关机：shutdown -c
[示例]从现在开始5分钟后关机：shutdown +5
[示例]20:30关机：shutdown 20:30
命令 Command
1.命令格式
格式：command [-options] [parameter]
说明：区分大小写!!!选项以-开头；参数不需要-；[]表示可选的；选项一定在参数前面；
2.常见命令
pwd：Print Work Directory，输出当前所在目录；使用最多；没有之一
ls：list[同dir]，显示当前或指定目录文件；
-a：隐藏文件
-l：列表显示详细信息
-R：recursion递归显示
-h：更加直观的显示文件大小
cd：Change Directory，更改工作目录；
cd：切换到用户主目录
cd ~：切换到用户主目录；也可以切换到用户主目录下某个文件夹，如cd ~/Desktop/
cd ..：切换到上级目录
cd .：切换到当前目录
cd -：在最近两次工作目录之间切换
touch：新建文件；鸿蒙系统的碰一碰；
文件不存在，则创建；文件存在，修改最近访问时间；
mkdir：make directory，创建工作目录；不允许同名；
-p：嵌套创建，如：
[示例] mkdir -p a/b/c/d
rm：remove，删除文件；不能恢复哦！！！
-r：删除非空目录；通常情况下，非空目录不能删除；
-f：强制删除，不给任何提示，即使不存在也会执行删除操作[简单粗暴型]；
clear：清屏；不能简写为cls哦；
[项目演练]根据学校、学院、年级、专业、班级的架构，在桌面Desktop创建对应的文件夹，并在班级文件夹中创建一个以学号+姓名命名的txt文件。验收后，逐级返回Home；
3.查看命令帮助
格式：command --help
格式：man command
说明：man是manual/手册；
4.通配符
操作[选择、删除等等]文件时，使用通配符更加方便、精准、高效；
* 任意个数的字符
? 任意一个字符
[abc] 匹配abc中任意一个字符
[a-b] -表示范围；匹配a到b中任意一个字符
[项目演练]
[示例] ls *3.txt：显示以3结束的所有txt文件；
[示例] ls *3*：显示以3结束的所有文件，类型不限；
[示例] ls ?2?.txt：文件名中间是2，长度为3的txt文件；
[示例] ls ?[123]?.txt：文件名中间是1或2或3，长度为3的txt文件；
[示例] ls ?[1-9]*.txt：文件名第2位是1-9某个数字的所有txt文件；
5.进阶命令
tree：树型显示当前或某个工作目录；
-d：只显示目录
如果不支持，请根据提示安装：sudo apt install tree
cp src des：copy，拷贝文件/目录；通常会带上相应的目录；如果复制时不想改文件名字，只需指定目标目录；
-i：information，如果有同名文件覆盖，则给出提示
-r：递归复制目录以及其中的内容，相当于整体复制文件夹
[示例] 拷贝到上级目录下的b目录，并更改名字：cp 123.txt ../b/321.txt
[示例] 拷贝到上级目录下的b目录，不更改名字：cp 123.txt ../b
[示例] 拷贝当前目录下目录b的内容到a目录里的c目录下；如果c目录不存在，则创建；如果c存在，则拷贝到c目录下：cp -r ./b a/c
mv src des：move，移动文件；通常会带上相应的目录；移动目录时，不需要额外的选项，如-r；可以在移动的过程中更改文件名；
用法1：移动文件
[示例] 移动b目录的112.txt到当前目录：mv b/112.txt .
用法2：移动文件的同时更改文件名
[示例] 移动b目录的122.txt到当前目录并改名221.txt：mv b/122.txt 221.txt
[示例] 为避免同名覆盖，可以带上-i参数，给出提示信息：mv -i 112.txt 114.txt
用法3：更改目录
[示例] 更改b目录为bb目录：mv b bb

历史命令：利用↑、↓方向键选择历史命令
CTRL+c返回命令状态

文件 File
1.文件类型
文件：白色
文件夹：蓝色
[项目演练] 创建隐藏文件：touch .123.txt并显示；
2.文件操作
cat：concatenate，查看文件内容、创建文件、追加内容等；
-b：仅非空行显示行号
-n：显示行号，包括空行
[示例] 显示文件121.txt内容：cat 121.txt
more：分页显示结果
[示例] 分屏显示文件121.txt内容，配合以下热键控制显示效果：more 121.txt
space 每按一次显示下一屏
enter 每按一次显示下一行
b 每按一次回滚一页
f 每按一次前滚一页
q 退出
grep：以高亮的形式显示文件中含有特定内容的行；相当于查找/搜索
-n：匹配行并显示行号
-v：排除内容，即：不包括指定内容的行，相当于取反搜索
-i：忽略大小写
^a：以a开头的行
a$：以a结束的行
[示例] 搜索文件中包含as的行：grep as 112.txt
[示例] 搜索文件中的不包括as的行并显示行号：grep -vn as 112.txt
[示例] 搜索文件中以as开头的行并显示行号：grep -n ^as 112.txt
[示例] 搜索文件中以as结束的行并显示行号：grep -n as$ 112.txt
echo：一般情况下，在终端输出内容；
>/>>：重定向，重新指定结果去向，即：将原本在终端显示的结果追加或覆盖到指定文件中，用于保存结果；如果文件不存在，则创建；可以不用空格分隔；相比touch创建文件，更加方便、直接；
[示例] 覆盖文件内容：echo hi,there >221.txt
[示例] 追加文件内容：echo hi,there >>221.txt
[示例] 将查看的结果放到文件中：ls -lh >112.txt
[示例] 将查看的结果追加到文件中：tree >>112.txt
|：管道，将一个命令的输出，作为另一个命令的输入，相当于多级过滤；通常配合more和grep命令使用；
[示例] 在显示主目录的结果中查找包含cc的行：ls -lha ~ | grep cc
[示例] 分屏显示主目录结果：ls -lha ~ | more
高级操作
1.SSH
Secure SHell：为远程管理服务器提供安全[加密]、高效[压缩]的数据链接；服务端口号默认是22；Linux默认已安装，window中需要单独安装客户端软件，如PuTTY、xShell等等；
格式：ssh [-p port] user@remote
