Axios

网络请求
以下测试用例基于json-server环境
简介 overview
异步请求；基于 promise 的 HTTP 库，可以用在浏览器和 node.js 中
通用、主流的第三方库；用于获取本地和远程json数据
默认是Promise用法；支持async/await用法
更多信息，点击访问Axios中文网
基本过程
1. 安装 - 在项目目录安装
>npm i axios --save
2. 引入 - 在项目中引入，并根据需要指定默认参数，如baseURL、params、method、header、timeout；baseURL应和数据服务器的地址保持一致
import axios from 'axios'
axios.defaults.baseURL = 'http://localhost:3000'
3. 使用 - 在组件的逻辑中请求数据，并应用到组件的结构中；资源服务器应提前开启，并严格按照接口使用

    

    
2. CDN方式 - 适合浏览器环境；建议将库下载到本地使用。。。CDN网络经常。。。

基本用例

  

    
获取 GET

    
可以传递参数；传递的参数在req.query中获取

    
1.无参

    

      
axios.get(baseURL + '/cont')
  .then(res => {
    console.log('res', res);
  })
  .catch(err => {
    console.log('err', err);
  })
  .finally(() => {
    console.log('done for log');
  })

    

    
2.有参 - 使用 params 提交

    

      
axios.get('/user/login', {
  params: {
    usn: user.value.usn,
    upass: user.value.upass
  }
})

    

    
3.有参 - 使用 ?拼接 提交

    

      
axios.get('/user/login?usn='+user.value.usn+'&upass='+user.value.upass)

    

    
增加 POST

    
默认是json，所以略去了请求头header

    

      
axios.post(baseURL + '/cont', {

  src: '1.jpg'
})
  .then(res => {
    console.log('res', res);

  })

请求方式

  

    
承诺 promise

    

      
axios.get('/user?ID=12345')
  .then((response) => {
    // 处理成功情况
    console.log(response);
  })
  .catch((error) => {
    // 处理错误情况
    console.log(error);
  })
  .finally(() => {
    // 总是会执行
})

    

    
异步等待 async/await

    

      
async function getUser() {
  try {
    const response = await axios.get('/user?ID=12345');
    console.log(response);
  } catch (error) {
    console.error(error);
  }
}
