JSON-Server

Overview Html Vue MP
快速搭建HTTP服务；需要 Node.js 环境支持
为应用提供JSON数据服务 - 仅仅是针对json文件的解析和应用
更多信息，请访问 NPM Json-Server；更多使用，请访问 从入门到膨胀
严格按照文档接口使用
方法
GET 数据获取|查询
配合id可以查询指定记录		 /posts
/posts/:id
POST 新增|提交数据；会自动增加id，也可以提交时，手动指定 /posts
PUT 以覆盖|替换的方式修改数据；配合id使用 /posts/:id
PATCH 以更新|部分修改的方式修改数据，配合id使用 /posts/:id
DELETE 删除数据，配合id使用 /posts/:id
安装依赖
通常使用命令行参数 -g 全局安装 JSON-server：可以在任何位置监听 json数据 请求 - 指哪打哪
任意位置打开 CMD 命令行窗口执行以下命令 - 确保已经安装了 Node
npm install json-server -g
如果 临时 使用，可以使用npx命令执行，不需要安装
npx json-server --watch data/students.json --port 3000
基本使用
创建数据文件夹 - 集中存放数据文件，如在D盘创建data文件夹
创建数据文件 - 后缀名为.json；早期数据交换多使用XML

应严格按照JSON文件的语法格式创建值对key-value；key必须用 双引号 包裹，value使用对应的数据类型即可；不允许有注释

在数据文件夹内创建数据文件，如students.json；根节点包括2个子节点|值对：digital和software；每个子节点使用数组保存信息节点。每个信息节点又是一个值对，包括用户id，姓名和星数；初始数据为空，通过添加和修改逐步增加；具体数据根据需求添加和修改 

{
  "digital": [
    {
      "uid": 20220110,
      "uname": "gl",
      "rank": 0
    }
  ],
  "software": [
    {
      "uid": 20220210,
      "uname": "gl",
      "rank": 0
    }
  ]
}
监听数据文件 - 在数据文件夹内，打开CMD控制台，监听文件，即开启数据服务

注意当前打开的CMD位置和监听数据文件的路径关系

系统自动检测并给出接口的地址；这个地址就是提供给项目使用的API接口

按住CTRL单击对应的地址，可以在浏览器中访问其中的内容；或直接在浏览器输入对应的地址

. 默认端口3000

json-server students.json

. 也可以指定端口

json-server students.json --port=5173

可以封装函数

})
func loadData(){
  axios.get('http://127.0.0.1:5173/software')
    .then(res => {
      //赋值
  })
}
数据渲染 - 判断数据是否为空
<div class="wrap">

      <div>

    </div>

  <template v-else>数据为空</template>

POST
增加数据；提交用户id，姓名和星数；完毕后重新拉取数据
function add() {
  axios.post('http://127.0.0.1:5173/software', {
    uid: uid.value,
    uname: uname.value,
    rank: rank.value
  })
    .then(res => {
      loadData()
    })
}
PATCH
修改数据；根据id或uid
由事件对象获取结构中绑定的数据
function mod(id, rank) {
  axios.patch('http://127.0.0.1:5173/software/' + id, {
    rank: rank + 1
  })
    .then(res => {
      //更新完毕后，创新加载数据
      loadData()
    })
}
DELETE
删除数据；根据id或uid
function del(id) {
  axios.delete('http://127.0.0.1:5173/software/' + id)
    .then(res => {
      loadData()
    })
}

根据用户id获取指定数据 - 用户提交前可以根据结果决定是不是已经提交；如果查询结果不为0，则已经提交，不允许再次提交
url: 'http://127.0.0.1:3000/user?id='+this.data.id
POST
提交数据；JSON-Server会自动增加id，类型字符串
wx.request({
  url: 'http://127.0.0.1:3000/msg',
  method:'POST',
  data:{
    id:2,
    uname:'nj'
  },
  success:res=>{
    console.log(res);
  }
})
DELETE
根据id删除数据；注意是直接拼接id
wx.request({
  url: 'http://127.0.0.1:3000/msg/'+e.currentTarget.dataset.id,
  method: 'DELETE',
  success: res => {
    console.log(res);
  },
  fail: err => {
    console.log(err);
  },
  complete: () => {
    console.log('done log');
  }
})
