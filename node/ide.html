切换镜像源 Registry
为了加快包的下载和使用，改善使用体验，通常需要切换镜像源
1. 查看 get 安装镜像源，默认是
>npm config get registry
https://registry.npmjs.org
2. 设置 set 为淘宝最新镜像源 - 感谢22软工的同学及时反馈
>npm config set registry https://registry.npmmirror.com
3. 再次查看安装镜像源
>npm config get registry
https://registry.npmmirror.com
安装依赖 Install
install可以简写为i
默认是生产环境

生产环境依赖 dependencies：项目上线后，所必须的依赖，如网络请求axios；使用--S/--save命令行参数

npm install package_name --save
npm install package_name --S

开发环境依赖 devDependencies：开发过程中需要的依赖，发布上线后，不再需要，如CSS预处理；使用--D/--save-dev命令行参数

npm install package_name --save-dev
npm install package_name --D
移除依赖
npm uninstall package_name

开放脚本执行权限
以管理员方式，打开PowerShell
查看权限 - 默认是禁止：Restricted
get-executionpolicy
修改权限为远程 remotesigned，按Y确认
set-executionpolicy remotesigned
再次查看权限，确认修改成功
