本地存储 LocalStorage

特点

以值对的形式存储在本地

是window的全局对象

持久化保存，除非用户主动清除

读取方便

存储容量大，约20MB

只能存字符串String！！！

不会出现在请求头中，可以节省带宽

不会被网络爬虫抓取

不同浏览器保存的数据不能共享

常用方法

设置值对localStorage.setItem(key,value)

根据key获取值对localStorage.getItem(key)

删除值对localStorage.removeItem(key)

清空数据localStorage.clear()

根据索引获取值对localStorage.key(n)

常用属性

保存数据的个数localStorage.length

查看保存的数据

F12 -> Application -> Storage -> Loacal Storage

提示：通过浏览器log的结果文字的颜色判断其类型：黑色为字符串；蓝色为数字

数值Number
      //设置
      localStorage.setItem('num', 2114)
      //获取
      let num = localStorage.getItem('num')
      console.log(num, Number(num));
字符串String
      //设置
      localStorage.setItem('str', 'male')
      //获取
      let str = localStorage.getItem('str')
      console.log(str);
布尔Boolean
      //设置
      localStorage.setItem('bool', true)
      //获取
      let bool = localStorage.getItem('bool')
      console.log(bool, Boolean(bool));
数组Array
      let arr = [1, 2, 3, 4, 5]
      localStorage.setItem('arr', arr)
      let tmpArr0 = localStorage.getItem('arr')
      console.log(tmpArr0);

      // 使用JSON.stringify转换为字符串
      localStorage.setItem('arr', JSON.stringify(arr))
      let tmpArr1 = localStorage.getItem('arr')
      console.log(tmpArr1);

      // 使用JSON.parse解析字符串
      let tmpArr2 = JSON.parse(localStorage.getItem('arr'))
      console.log(tmpArr2);
对象Object
      let obj = { name: 'glc' }
      localStorage.setItem('obj', obj)
      let obj0 = localStorage.getItem('obj')
      console.log(obj0);

      //使用JSON.stringify转换
      localStorage.setItem('obj', JSON.stringify(obj))
      // 使用JSON.parse解析
      let obj1 = JSON.parse(localStorage.getItem('obj'))
      console.log(obj1);

总结Summary

1.数据的设置和获取默认都是为字符串

2.如果存储的是数值型数据，获取时应使用Number()转换为数字

3.复合数据类型如数组和对象，保存时应使用 JSON.stringify() 转换；获取时应使用 JSON.parse() 解析；两者要配合使用

4.如果数据不存在，返回为null
