动画Animation

动画使用
复合写法
animation: name duration timing-function delay iteration-count direction fill-mode;
独立写法
animation-name动画名称
animation-duration持续时间
animation-timing-function变化效果
animation-delay延迟时间，必须是第2个时间参数
animation-iteration-count执行次数
animation-direction方向
animation-fill-mode结束模式
animation-play-state动画状态
动画定义

1.利用关键字：通常省略from，表示从当前状态开始

  @keyframes ani_name {
    from {      
    }

    to {      
    }
  }

2.利用百分比：可以定义多个百分比节点

  @keyframes ani_name {
    0% {      
    }

    100% {      
    }
  }
伸展
    .scale{
      animation: step 10s linear alternate infinite;
    }
步进
    //steps(n,end/start)；n：次数；默认是end，从状态结束位置开始
    .step {     
      animation: step 10s steps(10, start) infinite;
    }
    
    @keyframes step {
      to {
        width: 100%;
      }
    }
hover
    .hover {
      animation-play-state: paused;
    }
    
    .hover:hover {
      animation-play-state: running;
    }
focus
swiper
