课程表和简历

Front-End

前端开发基础

    主要涉及 HTML、CSS 和 JavaScript，用于构建网页结构、样式与交互功能

  • HTML 定义内容结构
  • CSS 控制布局与视觉效果
  • JavaScript 实现动态行为与用户交互

Vue.js

前端框架开发与应

    渐进式 JavaScript 框架，用于构建用户界面

  • 组件化开发模式
  • 支持双向数据绑定和虚拟 DOM
  • 易于集成与扩展
  • 适合开发单页面应用 SPA

Node.js

前端服务器

    一个基于 Chrome V8 引擎的 JavaScript 运行环境

  • 允许在服务器端执行 JavaScript
  • 采用事件驱动、非阻塞 I/O 模型
  • 适合构建高性能、可扩展的网络应用

Supabase

SaaS

    一个开源的后端即服务 BaaS 平台，提供数据库、身份验证、存储等功能，支持快速构建全栈应用

  • 基于 PostgreSQL
  • 具备实时功能与 RESTful API
  • 简化前后端开发流程

Mini Program

微信小程序

    一种无需安装、可快速启动的轻量级应用，基于微信平台运行

  • 支持跨平台开发
  • 结合 WXML、WXSS 和 JavaScript
  • 适用电商、工具、社交等多种场景
  • 提升用户体验与开发效率

Unity

虚拟现实开发

    一款跨平台游戏开发引擎，支持 2D、3D 游戏开发与虚拟现实 VR 应用构建

  • 提供可视化编辑器、物理系统与脚本控制
  • 脚本基于 JavaScript 或 C#
  • 广泛应用于游戏、动画、模拟与交互式内容开发

Python

编程语言

    一种简洁、易读的高级编程语言，支持多种编程范式

  • 语法清晰、简洁
  • 适用于数据分析、人工智能、Web 开发、自动化脚本等领域
  • 拥有丰富的库与活跃的社区支持

Ubuntu

操作系统

    基于 Linux 内核的开源操作系统

  • 稳定、安全和友好
  • 适用于服务器、桌面及云计算环境
  • 强大的社区支持与丰富的软件生态
  • 广泛用于开发与部署应用

Harmony OS

鸿蒙OS

    华为推出的分布式操作系统

  • 支持多设备协同与无缝体验
  • 具备低延迟、高安全性与跨平台能力
  • 适用于手机、平板、智能穿戴及物联网设备，提升全场景交互效率

Pattern

设计模式

    软件开发中常见问题的可复用解决方案

  • 提供代码结构优化与对象交互的最佳实践
  • 提升代码可维护性、扩展性与复用性
  • 包括创建型、结构型与行为型模式常见类型

Network

计算机网络

    通过通信设备与传输介质将地理位置不同的计算机系统相互连接，实现数据通信与资源共享

  • 由硬件、协议、拓扑结构组成
  • 支持互联网、局域网等多种应用场景

Office

大学计算机

    用于文档处理、表格计算、演示文稿等日常办公任务的软件套件

  • 包括 Word、Excel、PowerPoint
  • 提升文档创建、数据分析与信息展示的效率与专业性

Data Structures

数据结构

    计算机中组织和存储数据的方式，旨在提高数据访问与操作效率

  • 常见类型包括数组、链表、栈、队列、树、图
  • 是算法设计与程序开发的基础核心内容之一

Java

编程语言

    一种面向对象的高级编程语言

  • 跨平台、安全性高与性能稳定
  • 广泛应用于企业级应用、Android 开发、Web 服务与大数据处理
  • 支持多线程与自动内存管理

Operating System

操作系统

    管理计算机硬件与软件资源的核心程序

  • 提供用户界面、进程调度、内存管理与文件系统等功能
  • 常见类型包括 Windows、Linux、macOS 与 HarmonyOS
  • 支撑各类应用的运行与交互

WSN

无线传感网

    由大量微型传感器节点 Nodes 组成的自组织网络 Ad Hoc Network

  • 用于环境监测、数据采集与传输
  • 低功耗、低成本与高扩展性
  • 广泛应用于物联网、智能农业与军事侦察等领域
