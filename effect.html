::before ::after

说明

1.为元素指定伪元素而不增加DOM节点；多用于修饰元素，如JD购物车

2.::before ::after是子元素；可按照普通元素进行定位和布局等操作

3.内容content，通常放普通字符，也可利用attr（)获取元素属性

      dom::before {
        content: "";       
      }

box-shadow

说明

为元素添加一个或多个阴影。该属性是由逗号分隔的阴影列表，每个阴影由 2-4 个长度值、可选的颜色值以及可选的 inset 关键词来规定。省略长度的值是 0

语法 
  box-shadow: h-shadow v-shadow blur spread color inset;

filter

为元素添加滤镜效果；多个滤镜，请用空格分隔；原图如下；效果以动画形式呈现

1. blur()

模糊滤镜；模糊导致元素的发散会影响到周边其他元素；但是实际尺寸没有变化

    filter:blur(px)
2. drop-shadow()

投影滤镜：部分浏览器不支持第4个长度；如果添加，则不会呈现

    filter:drop-shadow(h-shadow v-shadow blur [spread] color)
3. grayscale()

将图像转换为灰阶

    filter:grayscale(%)
4. hue-rotate()

在图像上应用色相旋转。该值定义色环的度数。默认值为0deg，代表原始图像

    filter:hue-rotate(deg)
